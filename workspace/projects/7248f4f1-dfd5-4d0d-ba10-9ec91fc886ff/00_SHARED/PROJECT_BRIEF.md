# First Mission — Project Brief

Created: 2026-03-14T04:35:31.021742+00:00

## Problem
New users need to understand Porter through use, not through legacy internal project clutter.

## Solution
Provide a guided starter project where Porter can create workers, shape a plan, and produce artifacts safely.

## Scope
- one worker creation
- one directive
- one artifact or deliverable
- one transition into a real project

## Non-Negotiables
- keep it clean
- no internal Porter development residue
- explain through action

## Success Looks Like
The user understands how Porter orchestrates projects, workers, state, and artifacts.
