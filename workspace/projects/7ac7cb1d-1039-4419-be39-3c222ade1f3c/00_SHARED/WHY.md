# First Mission — Why

Created: 2026-03-20T11:29:55.126009+00:00

## Purpose
Teach a new operator how Porter actually works through one guided mission.

## Problem
New users should learn Porter from clear live workflow, not from legacy clutter or internal implementation residue.

## Outcome
The operator understands projects, workers, directives, state, and artifacts well enough to start a real project.
