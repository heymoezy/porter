  <behavior>
    - openSession(chatId, agentId, username, gatewayType, modelName, tokenBudget): creates a new active session row, returns the new session id
    - upsertSession(chatId, agentId, tokensToAdd, gatewayType, modelName, tokenBudget): finds the active session for (chatId OR agentId) and increments tokens_used, updates last_active_at, updates context_pct in metadata. If no active session exists, opens one. Returns { sessionId, tokensUsed, contextPct }.
    - rotateSession(sessionId): closes outgoing session (status='rotated', closed_at=now), reads bridge_dispatch_log for agentId over this session's lifetime to build a plain-text summary of what was worked on (top 3 distinct intents or task snippets, max 200 chars), writes a Recall concept (INSERT INTO concepts with source_type='session', trust_tier='medium', scope='agent', scope_id=agentId, confidence_score=60), opens a new session with same chat_id/agent_id/username/gateway_type/model_name/token_budget and tokens_used=0. Returns new session id.
    - getActiveSessions(): returns all session_registry rows where status='active', ordered by last_active_at DESC.
    - context_pct = tokens_used / token_budget (0 when token_budget=0 to avoid divide-by-zero). Stored in metadata.context_pct as float.
    - All DB ops use pool from ../../db/client.js. Import crypto from 'node:crypto' for randomUUID.
  </behavior>
  <action>
    Create backend/src/services/session-registry.ts.

    The file exports 4 functions: openSession, upsertSession, rotateSession, getActiveSessions.

    openSession(chatId: string | null, agentId: string | null, username: string, gatewayType: string, modelName: string, tokenBudget: number): Promise&lt;string&gt;
    - INSERT INTO session_registry (id, chat_id, agent_id, username, gateway_type, model_name, token_budget, tokens_used, context_msgs, status, metadata, created_at, last_active_at)
    - metadata = JSON.stringify({ context_pct: 0 })
    - Returns new id.

    upsertSession(chatId: string | null, agentId: string | null, tokensToAdd: number, gatewayType: string, modelName: string, tokenBudget: number): Promise&lt;{ sessionId: string; tokensUsed: number; contextPct: number }&gt;
    - SELECT id, tokens_used, token_budget, metadata FROM session_registry WHERE status='active' AND (chat_id=$chatId OR agent_id=$agentId) ORDER BY created_at DESC LIMIT 1
    - If no row: call openSession(...) then re-select.
    - UPDATE tokens_used = tokens_used + $tokensToAdd, context_msgs = context_msgs + 1, last_active_at = EXTRACT(EPOCH FROM NOW()), metadata = jsonb_set(metadata, '{context_pct}', to_jsonb((tokens_used + $tokensToAdd)::float / NULLIF(token_budget, 0)))
    - Compute contextPct = (tokensUsed + tokensToAdd) / Math.max(tokenBudget, 1)
    - Return { sessionId, tokensUsed: updatedCount, contextPct }

    rotateSession(sessionId: string): Promise&lt;string&gt;
    - SELECT agent_id, chat_id, username, gateway_type, model_name, token_budget FROM session_registry WHERE id=$sessionId
    - UPDATE session_registry SET status='rotated', closed_at=EXTRACT(EPOCH FROM NOW()) WHERE id=$sessionId
    - Build summary: SELECT DISTINCT COALESCE(intent, LEFT(message_text, 80)) AS snippet FROM bridge_dispatch_log WHERE agent_id=$agentId AND created_at >= $sessionCreatedAt ORDER BY created_at ASC LIMIT 5
    - Combine snippets into: "Session summary: worked on [snippet1]; [snippet2]; ..." (truncate at 300 chars)
    - INSERT INTO concepts with memory_kind='concept', trust_tier='medium', scope='agent', scope_id=agentId, content=summary, source_type='session', source_url=null, confidence_score=60, session_id=sessionId
    - Call openSession with same chat_id/agent_id/username/gateway_type/model_name/token_budget
    - Return new session id.

    getActiveSessions(): Promise&lt;Array&lt;{ id, chat_id, agent_id, username, gateway_type, model_name, tokens_used, token_budget, context_pct, created_at, last_active_at }&gt;&gt;
    - SELECT id, chat_id, agent_id, username, gateway_type, model_name, tokens_used, token_budget, metadata->>'context_pct' AS context_pct, created_at, last_active_at FROM session_registry WHERE status='active' ORDER BY last_active_at DESC

    ```typescript
    try {
      const totalTokens = (result.inputTokens ?? 0) + (result.outputTokens ?? 0);
      if (totalTokens > 0 && (ctx.chatId || ctx.agentId)) {
        // Resolve context_window from models table for this gateway+model
        let tokenBudget = 0;
        try {
          const { rows: mrows } = await pool.query<{ context_window: number }>(
            `SELECT context_window FROM models WHERE gateway_id = $1 AND model_name = $2 LIMIT 1`,
            [decision.gatewayRow.id, decision.modelName],
          );
          tokenBudget = mrows[0]?.context_window ?? 0;
        } catch { /* non-fatal */ }

        await upsertSession(
          ctx.chatId ?? null,
          ctx.agentId ?? null,
          totalTokens,
          decision.gatewayRow.type,
          decision.modelName,
          tokenBudget,
        );
      }
    } catch { /* non-critical — never block dispatch */ }
    ```
