    ```typescript
    export async function buildMemoryContext(opts: {
      agentId?: string;
      projectId?: string;
      tokenBudget?: number;
      searchQuery?: string;
    }): Promise<string>
    ```

    **Token estimation helper** (internal, not exported):
    ```typescript
    function estimateTokens(text: string): number {
      return Math.ceil(text.length / 4);
    }
    ```

    **Implementation — 5-tier pipeline:**

    Default tokenBudget = 2000. Track `remainingTokens` starting at the budget value.

    **Tier 1: Agent identity (~200 tokens)**
    If agentId is provided:
    - Query: `SELECT name, role, config FROM personas WHERE id = $1`
    - Extract name, role from result
    - Check config for memory_token_budget — if present, override the budget
    - Format: `## Agent Identity\nName: {name}\nRole: {role}\n`
    - Subtract estimated tokens from remaining
    - If no agentId, skip this tier entirely

    **Tier 2: Directives (~300 tokens)**
    Query active directives, including project-scoped if projectId provided:
    - If projectId: `SELECT content, priority FROM directives WHERE status = 'active' AND (scope = 'workspace' OR (scope = 'project' AND scope_id = $1)) ORDER BY priority ASC`
    - If no projectId: `SELECT content, priority FROM directives WHERE status = 'active' AND scope = 'workspace' ORDER BY priority ASC`
    - Format as `## Directives\n` + each directive content separated by newlines
    - Add directives one by one until remaining tokens are consumed; stop when adding the next would exceed budget
    - Subtract tokens consumed

    **Tier 3: Project notes (~400 tokens)**
    Only if projectId is provided:
    - Query: `SELECT content, note_type, confidence_score FROM project_notes WHERE project_id = $1 AND status = 'active' ORDER BY confidence_score DESC`
    - Format as `## Project State\n` + each note as `[{note_type}] {content}`
    - Add notes one by one until remaining tokens exhausted
    - Subtract tokens consumed

    **Tier 4: Agent notes (~400 tokens)**
    Only if agentId is provided:
    - Query: `SELECT content, note_type, confidence_score FROM agent_notes WHERE agent_id = $1 AND status = 'active' ORDER BY confidence_score DESC`
    - Format as `## Agent Knowledge\n` + each note as `[{note_type}] {content}`
    - Add notes one by one until remaining tokens exhausted
    - Subtract tokens consumed

    **Tier 5: Archival FTS search (~remaining tokens)**
    Only if searchQuery is provided AND remainingTokens > 50:
    - Query: `SELECT content, confidence_score FROM concepts WHERE search_vector @@ websearch_to_tsquery('english', $1) AND status = 'active' ORDER BY ts_rank(search_vector, websearch_to_tsquery('english', $1)) DESC LIMIT 10`
    - Format as `## Related Knowledge\n` + each concept content
    - Add concepts one by one until remaining tokens exhausted

    **Return:** Concatenate all non-empty sections with `\n\n` separator. If all sections are empty, return empty string ''.

    Wrap entire function body in try/catch — on any error, log to console.error and return '' (never fail the caller).
