    **Step 4: Create retry.ts**
    File: `backend/src/services/bridge/retry.ts`
    - Import `ErrorClass` from './types.js'
    - Export `classifyError(err: Error): ErrorClass` — regex-based classification:
      - transient: /429|rate.?limit|too.?many|503|service.?unavailable/i
      - configuration: /401|403|unauthorized|forbidden/i
      - persistent: everything else (500, ECONNREFUSED, timeout, etc.)
    - Export `isTransientError(err: Error): boolean` — returns `classifyError(err) === 'transient'`
    - Export `async function withRetry<T>(fn: () => Promise<T>, maxAttempts = 3, baseDelayMs = 1000): Promise<T>`
      - Loop 1..maxAttempts
      - On catch: classify error. If NOT transient, throw immediately
      - If transient and not last attempt: `await new Promise(r => setTimeout(r, baseDelayMs * 2^(attempt-1)))`
      - After all attempts exhausted, throw lastError
