  <action>
    Create `backend/src/routes/v1/connections.ts` as a Fastify plugin with these endpoints:

    **GET /** — List all workspace connections
    - Auth: requireAuth (any authenticated user)
    - Query: `SELECT * FROM workspace_connections ORDER BY provider ASC`
    - Response: `success(rows)` — NEVER include decrypted meta_json in list response. Return rows with `meta_json: '[encrypted]'` for encrypted rows (meta_encrypted=1), or the raw meta_json for unencrypted rows.
    - Non-admin users: same response (read access allowed per CONTEXT.md)

    **GET /:id** — Get single connection detail
    - Auth: requireAuth
    - Query by id
    - Return 404 if not found
    - Do NOT decrypt meta_json — credentials never sent to frontend

    **POST /** — Create connection
    - Auth: requireAuth + admin role check. If `request.user.role !== 'admin'`, return `fail('Admin access required', 403)`
    - Zod body schema: `{ provider: z.string(), kind: z.enum(['oauth2', 'api_key']), display_name: z.string().optional(), meta_json: z.record(z.string(), z.unknown()).optional(), scopes: z.array(z.string()).optional() }`
    - Generate UUID for id
    - If meta_json provided: encrypt with `encryptCredential(JSON.stringify(body.meta_json))`, set `meta_encrypted = 1`
    - INSERT into workspace_connections with `installed_by = request.user.username`
    - Return `success({ id, provider, status: 'disconnected' })`

    **PUT /:id** — Update connection (credentials, display name)
    - Auth: admin only
    - Zod body: `{ display_name: z.string().optional(), meta_json: z.record(z.string(), z.unknown()).optional(), status: z.enum(['connected', 'disconnected', 'needs_reauth', 'error']).optional() }`
    - If meta_json provided: encrypt and set meta_encrypted=1
    - UPDATE workspace_connections SET ... WHERE id = :id, updated_at = unixepoch('now')
    - Return 404 if no row affected

    **DELETE /:id** — Remove connection
    - Auth: admin only
    - DELETE FROM workspace_connections WHERE id = :id
    - Also DELETE FROM project_connections WHERE connection_id = :id (cascade)
    - Return `success({ deleted: true })`
    - Emit SSE: `connection:status` with `{ provider, status: 'disconnected' }`

    **GET /project/:projectId** — List connections for a project
    - Auth: requireAuth
    - Query: `SELECT wc.*, pc.access_mode, pc.status as override_status FROM workspace_connections wc LEFT JOIN project_connections pc ON pc.connection_id = wc.id AND pc.project_id = :projectId`
    - Return all workspace connections with project-level override data attached

    **POST /project/:projectId** — Attach workspace connection to project
    - Auth: requireAuth (operators can manage project overrides per CONTEXT.md)
    - Zod body: `{ connection_id: z.string(), access_mode: z.enum(['read', 'read_write']).default('read') }`
    - INSERT OR REPLACE into project_connections
    - Return `success({ attached: true })`

    **DELETE /project/:projectId/:connectionId** — Detach connection from project
    - Auth: requireAuth
    - DELETE FROM project_connections WHERE project_id = :projectId AND connection_id = :connectionId
    - Return `success({ detached: true })`

    Import emitSSE from scheduler.ts for SSE events. Import encryptCredential from credential-crypto.ts. Use sqlite.prepare() for all queries (consistent with codebase pattern).
