    Create `backend/src/services/github.ts`:

    Import `Octokit` from `octokit` (the batteries-included v5 package). Import `decryptCredential` from `../lib/credential-crypto.js`. Import `sqlite` from `../db/client.js`.

    **getGitHubClient(connectionId?: string): Octokit**
    - If connectionId provided, query workspace_connections by id. Otherwise, query WHERE provider='github' AND status='connected' LIMIT 1.
    - If no connection found, throw `Error('No GitHub connection found — connect via Connections page')`
    - Decrypt meta_json: `const creds = JSON.parse(decryptCredential(row.meta_json))`
    - Return `new Octokit({ auth: creds.access_token })`
    - NEVER log the access_token value

    **listRepos(connectionId?: string): Promise<{ name: string; full_name: string; private: boolean; html_url: string }[]>**
    - Get octokit client
    - Call `octokit.rest.repos.listForAuthenticatedUser({ per_page: 100, sort: 'updated' })`
    - Return mapped array with name, full_name, private, html_url

    **readFile(owner: string, repo: string, path: string, ref?: string, connectionId?: string): Promise<{ content: string; sha: string }>**
    - Get octokit client
    - Call `octokit.rest.repos.getContent({ owner, repo, path, ref })`
    - Decode Base64 content: `Buffer.from(data.content, 'base64').toString('utf8')`
    - Return { content, sha }

    **createBranch(owner: string, repo: string, branchName: string, fromRef?: string, connectionId?: string): Promise<string>**
    - Get octokit client
    - Get SHA of source ref (default 'main'): `octokit.rest.git.getRef({ owner, repo, ref: 'heads/' + (fromRef ?? 'main') })`
    - Create ref: `octokit.rest.git.createRef({ owner, repo, ref: 'refs/heads/' + branchName, sha })`
    - Return the new branch name

    **createPullRequest(owner: string, repo: string, params: { title: string; body: string; head: string; base?: string }, connectionId?: string): Promise<{ number: number; html_url: string }>**
    - Get octokit client
    - Call `octokit.rest.pulls.create({ owner, repo, title: params.title, body: params.body, head: params.head, base: params.base ?? 'main' })`
    - Return { number, html_url }

    Export all functions. Each function takes an optional connectionId parameter to support project-level overrides (different GitHub account per project).

    Error handling: wrap all octokit calls in try/catch. If 401 error, update connection status to 'needs_reauth' and emit SSE `connection:status`. Re-throw with descriptive message.
