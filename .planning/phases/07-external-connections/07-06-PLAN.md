  <action>
    Create `backend/src/services/email.ts`:

    Import `nodemailer` from `nodemailer`, `{ ImapFlow }` from `imapflow`, `decryptCredential` from `../lib/credential-crypto.js`, `sqlite` from `../db/client.js`, `{ emitSSE }` from `./scheduler.js`, `crypto` from `crypto`.

    **getEmailCredentials(connectionId?: string)**: Reads workspace_connections WHERE provider='email' AND status='connected'. Decrypts meta_json. Returns `{ access_token, refresh_token, email }`. Throws if no connection.

    **sendEmail(params: { to: string; subject: string; body: string; agentName?: string; connectionId?: string }): Promise<{ messageId: string }>**
    - Get credentials via getEmailCredentials
    - Create nodemailer transport:
      ```typescript
      const transport = nodemailer.createTransport({
        service: 'gmail',
        auth: {
          type: 'OAuth2',
          user: creds.email,
          accessToken: creds.access_token,
          clientId: process.env.GOOGLE_CLIENT_ID,
          clientSecret: process.env.GOOGLE_CLIENT_SECRET,
          refreshToken: creds.refresh_token,
        },
      });
      ```
    - Set `from` field: if agentName provided, use `"${agentName} via Porter <${creds.email}>"`, else `"Porter <${creds.email}>"`
    - Call `transport.sendMail({ from, to, subject, html: body })`
    - Return `{ messageId: info.messageId }`
    - On auth error (token expired): update connection status to 'needs_reauth', emit SSE

    **routeInboundEmail(from: string, subject: string, body: string, projectId?: string): string | null**
    - Check routing rules stored in workspace_connections meta_json under `routing_rules` key
    - Rule format: `{ pattern: string, agent_id: string }` — pattern matched against `from` address
    - If rule matches: insert agent_job with trigger_type='email_received', return agent_id
    - If no rule matches: insert agent_job for Porter (master agent) with prompt containing the email content for AI-based routing
    - Return agent_id that was dispatched to, or null if no connection

    **startImapIdle(connectionId?: string): Promise<void>**
    - Get credentials
    - Create ImapFlow client:
      ```typescript
      const client = new ImapFlow({
        host: 'imap.gmail.com',
        port: 993,
        secure: true,
        auth: {
          user: creds.email,
          accessToken: creds.access_token,
        },
        logger: false,
      });
      ```
    - Connect and get INBOX lock
    - Listen for 'exists' event (new message):
      - Fetch latest message: `client.fetchOne('*', { source: true, envelope: true })`
      - Extract from, subject, text body
      - Call `routeInboundEmail(from, subject, body)`
      - Emit SSE: `agent:activity` with email trigger summary
    - Call `client.idle()` — blocks until server sends notification
    - Wrap entire flow in reconnect loop: on 'close' event, wait 5s, reconnect. Max 3 consecutive failures before marking connection 'degraded'.
    - Store client reference in module-level variable for cleanup

    **stopImapIdle(): void**
    - If IMAP client exists, call `client.logout()` and set to null

    Export all 4 functions.
  </action>
