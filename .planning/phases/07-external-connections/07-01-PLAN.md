    - `decryptCredential(encoded: string): string`: splits on ':', creates decipher, sets auth tag, decrypts. Returns plaintext string. Throws on tampered data (GCM auth failure).
    - `validatePorterSecret(): boolean`: returns true if PORTER_SECRET is set and non-empty, false otherwise. Does NOT throw.
