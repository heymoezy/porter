   Component `WatchersPage`:
   - Use `useQuery` from @tanstack/react-query to fetch `api<{ watchers: WatcherRow[]; total: number }>("/api/v1/admin/watchers?limit=100")`
   - Auto-refresh every 30s: `refetchInterval: 30_000`
   - Page header: "Project Watchers" with subtitle showing total active count
   - Filter bar: status dropdown (All / Active / Paused / Error), project dropdown (populated from unique project_names in response)
   - Client-side filtering (data set is small enough — <100 watchers expected)
   
   Table columns:
   - **Name** — watcher name, bold
   - **Project** — project_name, muted text
   - **Type** — watcher_type with color-coded badge:
     - web_search: blue badge "Web"
     - rss_feed: orange badge "RSS"
     - email_monitor: purple badge "Email"
     - custom: gray badge "Custom"
   - **Status** — color-coded pill:
     - active: green dot + "Active"
     - paused: yellow dot + "Paused"
     - error: red dot + "Error" (hover shows last_error)
   - **Last Run** — relative time from last_run_at (e.g. "3h ago", "never")
   - **Next Run** — relative time until next_run_at (e.g. "in 2h", "overdue" if past)
   - **Runs** — run_count number
   - **Findings** — finding_count number
   - **Schedule** — human-readable interval (e.g. "Every 6h", "Every 24h", "Hourly")
   
   Helper function `formatRelativeTime(epoch: number | null): string`:
   - null -> "never"
   - Compute diff from Date.now()/1000
   - < 60s: "just now"
   - < 3600: "{M}m ago" / "in {M}m"
   - < 86400: "{H}h ago" / "in {H}h"
   - else: "{D}d ago" / "in {D}d"
   
   Helper function `formatInterval(seconds: number): string`:
   - 3600 -> "Hourly"
   - 21600 -> "Every 6h"
   - 43200 -> "Every 12h"
   - 86400 -> "Daily"
   - else -> "Every {N}h"
   
   Empty state: "No watchers configured. Create watchers via the API or project settings."
   
   Error state: Show error message with retry button.
   
   Styling: Use Tailwind classes consistent with bridge.tsx patterns — dark surface bg (bg-[var(--surface)]), border colors (border-[var(--border)]), text colors (text-[var(--text)], text-[var(--text)]/60 for muted).
