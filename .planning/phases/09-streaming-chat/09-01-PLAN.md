Create `backend/src/services/stream-service.ts` with:

1. **StreamBackend interface:**
```typescript
export interface StreamBackend {
  name: string;
  stream(prompt: string, signal: AbortSignal): AsyncIterable<string>;
}
```

2. **OllamaStreamBackend class** implementing StreamBackend:
   - `name = 'ollama'`
   - `async *stream(prompt, signal)` that:
     - Calls `fetch('${config.ollamaUrl}/api/generate', { method: 'POST', headers: {'Content-Type':'application/json'}, body: JSON.stringify({model: config.ollamaModel, prompt, stream: true}), signal })`
     - Reads response body as ReadableStream via `resp.body.getReader()`
     - Parses NDJSON lines: each line is `{"response":"token","done":false}` or `{"response":"","done":true,...}`
     - Yields `chunk.response` for each non-empty response
     - Returns when `chunk.done === true`
     - Buffer handling: accumulate partial lines with TextDecoder, split on `\n`, keep incomplete tail in buffer

3. **OpenClawStreamBackend class** implementing StreamBackend:
   - `name = 'openclaw'`
   - `async *stream(prompt, signal)` that:
     - Calls `fetch('${config.porterPyUrl}/api/dispatch', { method: 'POST', headers: {'Content-Type':'application/json'}, body: JSON.stringify({persona_id: 'porter', message: prompt}), signal })`
     - Awaits full text response
     - Word-chunks the response: split on spaces, accumulate chunks of ~25+ chars, yield each chunk
     - Between chunks: `await new Promise(resolve => setTimeout(resolve, 15))` for typewriter pacing
     - Check `signal.aborted` before each yield, return early if aborted

4. **selectStreamBackend function:**
```typescript
export function selectStreamBackend(message: string, backendHint?: 'ollama' | 'openclaw' | 'auto'): StreamBackend {
  if (backendHint === 'ollama') return new OllamaStreamBackend();
  if (backendHint === 'openclaw') return new OpenClawStreamBackend();
  // 'auto' or undefined: use shouldRouteCheap from ai-router
  return shouldRouteCheap(message) ? new OllamaStreamBackend() : new OpenClawStreamBackend();
}
```
