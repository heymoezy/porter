## Decisions Made

See `key-decisions` frontmatter for the full list. Highlights:

- **Hook absence is graceful-skip not failure.** Plan 05 ships in Wave 1 alongside Plan 01 (disjoint files). Plan 03 ships the hooks in a later wave. The graceful-skip envelope means the smoke can run today (Wave 1) and skip TRC-02/03/08 cleanly, then turn those skips → green automatically once Plan 03 deploys the hooks.
- **Backend reachability is a soft gate.** Schema checks (TRC-01) run via psql alone and don't need Fastify. The script exits 0 with a warn line if `/health` is unreachable AND TRC-01 passed — this lets the migration alone be validated before the API surface ships.
- **Poll-with-timeout, never fixed sleep.** Per F-5 revision. 20 × 0.5s = 10s ceiling per assertion; early-break on success. Tolerates async fire-and-forget round-trips and backend INSERT latency without flakiness.
