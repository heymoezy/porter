3. **Export async function runAtlasCheck(projectId: string, fsPath: string, projectName: string): Promise<AtlasFinding[]>**
   - findings: AtlasFinding[] = []
   - Verify fsPath exists (fs.access). If not, return [{ type: 'missing_dir', path: fsPath, detail: 'Project root directory missing' }]

   - **Check canonical directories:**
     For each dir in CANONICAL_DIRS:
       - Check if dir exists at fsPath/dir
       - If missing: create it with fs.mkdir(path.join(fsPath, dir), { recursive: true })
       - Push finding: { type: 'repaired', path: dir, detail: `Recreated missing directory /${dir}/` }

   - **Check _system files:**
     For each expected file in ['project.md', 'checkpoint.md', 'memory.md', 'decisions.md', 'tasks.md', 'agents.md']:
       - Check if file exists at fsPath/_system/{file}
       - If missing: push finding { type: 'missing_file', path: `_system/${file}`, detail: `System file missing: ${file}` }
       - Do NOT recreate missing _system files (they may have been intentionally removed — only flag, don't auto-repair content files)

   - **Check for misplaced files in root:**
     - Read fsPath directory entries (fs.readdir with withFileTypes)
     - For each entry that is a FILE (not directory):
       - Push finding: { type: 'misplaced_file', path: entry.name, detail: `File "${entry.name}" found in project root — should be in a subdirectory (intake/, context/, work/, outputs/, or archive/)` }
     - Skip hidden files (starting with '.') and common root files like README.md, .gitignore

   - Return findings

4. **async function logAtlasFindings(projectId: string, projectName: string, findings: AtlasFinding[]): Promise<void>**
   - If findings.length === 0, return (no activity for clean projects)
   - Group findings by type for summary
   - Build summary: "Atlas: {repaired_count} repaired, {missing_count} missing, {misplaced_count} misplaced files"
   - INSERT INTO agent_activity with:
     - agent_id: 'atlas'
     - job_id: null
     - project_id: projectId
     - event_type: 'atlas_check'
     - summary: the summary string
     - detail: JSON.stringify({ findings, checked_at: Date.now() / 1000 })
   - emitSSE('project:activity', { projectId, eventType: 'atlas_check', findingCount: findings.length })
   - Best-effort: catch errors and log

5. **Export async function scheduleAtlasRuns(): Promise<void>**
   - Query all projects with fs_path IS NOT NULL AND status = 'active':
     ```sql
     SELECT id, name, fs_path FROM projects WHERE fs_path IS NOT NULL AND status = 'active'
     ```
   - For each project: run runAtlasCheck(id, fs_path, name), then logAtlasFindings
   - Wrap each project check in try/catch (one project failure doesn't stop others)
   - Log total: "[atlas] checked {N} projects, {totalFindings} findings"
