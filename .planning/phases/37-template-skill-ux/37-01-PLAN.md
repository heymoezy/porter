    - Return: `ok({ template_id: id, skills: rows })`

    **Endpoint 2: POST /:id/skills** (TUX-02)
    - Body: `{ skill_id: string, is_mandatory?: number, assignment_rationale?: string }`

    - Return: `ok({ attached: true, template_id: id, skill_id })`

    **Endpoint 3: DELETE /:id/skills/:skillId** (TUX-02)

    - Return: `ok({ detached: true, template_id: id, skill_id: skillId })`
