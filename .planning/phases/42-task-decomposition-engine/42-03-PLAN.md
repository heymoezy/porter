    **Part A: Create decomposition-engine.ts**

    This is the single entry point that orchestrates the full pipeline:

    1. **decomposeAndExecute(message: string, opts: { projectId?: string; chatId?: string; userId?: string }): Promise<{ rootId: string; taskCount: number }>**
       - Generate rootId via uuidv4()
       - Load available agents for the project (if projectId): query personas table for agents linked to this project
       - If no project agents, use default empty array (planner handles this gracefully)
       - Call planTasks({ message, projectId, chatId, availableAgents })
       - Call insertTaskTree(rootId, plan, request)
       - Broadcast SSE: `decomposition:started` with { rootId, taskCount, tasks: [{id, description, deps}] }
       - Fire-and-forget the execution + join pipeline (do NOT await):
         ```typescript
         void (async () => {
           try {
             await executeTaskTree(rootId);
             const joined = await joinResults(rootId);
             // Update root node with synthesis result
             await pool.query(
               `UPDATE task_nodes SET status = $1, result = $2, completed_at = EXTRACT(EPOCH FROM NOW()) WHERE id = $3`,
               [joined.action === 'synthesized' || joined.action === 'partial' ? 'completed' : 'failed',
                JSON.stringify({ synthesis: joined.response, action: joined.action }),
                rootId]
             );
             // If replan, attempt ONE replan cycle
             if (joined.action === 'replan') {
               // Replan: cancel failed nodes, re-plan with error context, re-execute
               // (bounded: only 1 replan attempt, then fail)
               broadcast('decomposition:replan', { rootId, failedTasks: joined.failedTasks });
               // ... replan logic (plan again, insert new nodes, execute again) ...
               // For v1: just mark as failed with replan note
               await pool.query(
                 `UPDATE task_nodes SET status = 'failed', error = 'Replan needed — manual intervention required' WHERE id = $1`,
                 [rootId]
               );
             }
             // Save synthesized response to chat history (if chatId)
             if (opts.chatId && joined.response) {
               await pool.query(
                 `INSERT INTO chat_messages (chat_id, role, content, model_id, timestamp) VALUES ($1, 'assistant', $2, 'porter-synthesis', NOW())`,
                 [opts.chatId, joined.response]
               );
             }
             broadcast('decomposition:complete', { rootId, chatId: opts.chatId, response: joined.response, action: joined.action });
           } catch (err) {
             const errorMsg = err instanceof Error ? err.message : String(err);
             await pool.query(
               `UPDATE task_nodes SET status = 'failed', error = $1, completed_at = EXTRACT(EPOCH FROM NOW()) WHERE id = $2`,
               [errorMsg, rootId]
             );
             broadcast('decomposition:failed', { rootId, error: errorMsg });
           }
         })();
         ```
       - Return immediately: `{ rootId, taskCount: plan.tasks.length }`

    // Only classify for messages that have a chat context (not warm-up calls)
    // and are not already in a decomposition context
    const classification = await classify(message);

    if (classification.classification === 'complex') {
      // Route to decomposition engine — async execution, immediate ack
      try {
        const { rootId, taskCount } = await decomposeAndExecute(message, {
          projectId: projectId ?? undefined,
          chatId: chatId ?? undefined,
          userId: request.sessionUser!.username,
        });

        // Send decomposition acknowledgment as SSE
        reply.raw.write(`data: ${JSON.stringify({

          decomposition: true,
          rootId,
          taskCount,
        })}\n\n`);

        // Save user message to chat history
        if (chatId) {
          const existingChat = (await pool.query('SELECT id FROM chats WHERE id = $1', [chatId])).rows[0];
          if (!existingChat) {
            await pool.query(
              'INSERT INTO chats (id, title, model_id, username, project_id, created_at, updated_at) VALUES ($1, $2, $3, $4, $5, NOW(), NOW())',
              [chatId, message.slice(0, 50), 'porter-decompose', request.sessionUser!.username, projectId ?? null]
            );
          }
          await pool.query(
            'INSERT INTO chat_messages (chat_id, role, content, model_id, timestamp) VALUES ($1, $2, $3, $4, NOW())',
            [chatId, 'user', message, null]
          );
        }

        // Signal done — the decomposition engine will send results via SSE events
        reply.raw.write(`data: ${JSON.stringify({
          done: true,
          backend: 'porter-decompose',
          full_response: `Decomposing into ${taskCount} subtasks... Progress will appear in real-time.`,

        })}\n\n`);
        reply.raw.end();
        return reply;
      } catch (decompErr) {
        // Decomposition failed — fall through to normal dispatch
        // Log but don't block the user
        console.error('[TDE] Decomposition failed, falling through to direct dispatch:', decompErr);
        // Continue to selectStreamBackend below
      }
    }
    // If simple or decomposition failed, continue with existing direct dispatch flow...
