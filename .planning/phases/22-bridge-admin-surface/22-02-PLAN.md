Add `POST /gateways` handler inside the existing plugin function, after the GET endpoints.

The handler reads `req.body` as `{ action, ...body }` where action is one of: 'add', 'update', 'remove', 'validate'.

**action = 'remove':**
Require `body.id`. Execute `DELETE FROM gateways WHERE id = $1` (ON DELETE CASCADE handles gateway_credentials and models per Pitfall 2). Return `ok({ removed: true, id: body.id })`.

**action = 'validate':**
Require `body.id`. SELECT gateway row by id. If not found, return `ok({ valid: false, error: 'NOT_FOUND' })`. Map through `mapRawToGatewayRow()`, call `createAdapter(gatewayRow)`. If no adapter, return `ok({ valid: false, error: 'NO_ADAPTER' })`. Call `adapter.health()`. Return `ok({ valid: health.healthy, latencyMs: health.latencyMs, error: health.error ?? null })`.

**action = 'add':**
Require `body.type`, `body.name`. Validate type against VALID_GATEWAY_TYPES — if invalid, return `err('INVALID_TYPE', 'type must be one of: ' + [...VALID_GATEWAY_TYPES].join(', '))`.
Generate `id = crypto.randomUUID()`.
INSERT INTO gateways with columns: id, type, name, url (body.url ?? null), auth_method (body.auth_method ?? 'none'), status ('unavailable'), source ('manual'), priority (body.priority ?? 50), capabilities (body.capabilities ?? '[]' as JSONB), metadata (body.metadata ?? '{}' as JSONB), enabled (body.enabled ?? 1), created_at and updated_at as `EXTRACT(EPOCH FROM NOW())`.

If `body.api_key` is provided:
- Call `validatePorterSecret()` — if false, return `err('SECRET_MISSING', 'PORTER_SECRET not configured — cannot encrypt credentials')`.
- Generate credential ID: `crypto.createHash('sha256').update(body.api_key).digest('hex').slice(0, 36)`.
- `const encrypted = encryptCredential(body.api_key)`.
- `const masked = '***' + body.api_key.slice(-4)`.
- INSERT INTO gateway_credentials with ON CONFLICT (id) DO UPDATE for idempotent upserts.

Return `ok({ created: true, id })`.

**action = 'update':**
Require `body.id`. Build a dynamic SET clause from allowed fields: name, url, auth_method, priority, capabilities, metadata, enabled. Only include fields that are present in body (not undefined). Always set `updated_at = EXTRACT(EPOCH FROM NOW())`.
Execute `UPDATE gateways SET ... WHERE id = $N`.
If `body.api_key` is provided, handle credential upsert same as 'add' action.
Return `ok({ updated: true, id: body.id })`.

**Default (unknown action):**
Return `err('INVALID_ACTION', 'action must be one of: add, update, remove, validate')`.

**POST `/routing-rules` (ADM-06):**
Add handler that reads `req.body` as `{ action, ...body }` where action is: 'create', 'update', 'delete', 'list'.

Define validation sets:
```typescript
const VALID_SCOPES = new Set(['global', 'agent', 'project', 'gateway']);
const VALID_RULE_ACTIONS = new Set(['force_model', 'block_gateway', 'cap_cost_usd', 'prefer_local']);
```

**action = 'list':**
```sql
SELECT id, scope, scope_id, action, action_value, enabled, priority, description, created_by, created_at, updated_at
FROM routing_rules
ORDER BY priority ASC, created_at ASC
```
Return `ok({ rules: rows })`.

**action = 'create':**
Validate `body.scope` against VALID_SCOPES — if invalid, return `err('INVALID_SCOPE', 'scope must be one of: ' + [...VALID_SCOPES].join(', '))`.
Validate `body.action_type` against VALID_RULE_ACTIONS — if invalid, return `err('INVALID_ACTION_TYPE', 'action_type must be one of: ' + [...VALID_RULE_ACTIONS].join(', '))`.
Generate `id = crypto.randomUUID()`.
INSERT INTO routing_rules with columns: id, scope, scope_id (body.scope_id ?? null), action (body.action_type), action_value (body.action_value ?? null), enabled (body.enabled ?? 1), priority (body.priority ?? 50), description (body.description ?? null), created_by (request.sessionUser?.username ?? null), created_at and updated_at as `EXTRACT(EPOCH FROM NOW())`.
Return `ok({ created: true, id })`.

**action = 'update':**
Require `body.id`. Build dynamic SET clause from allowed fields: scope, scope_id, action (from body.action_type), action_value, enabled, priority, description. Always set `updated_at = EXTRACT(EPOCH FROM NOW())`.
If scope provided, validate against VALID_SCOPES.
If action_type provided, validate against VALID_RULE_ACTIONS.
Execute UPDATE. Return `ok({ updated: true, id: body.id })`.

**action = 'delete':**
Require `body.id`. DELETE FROM routing_rules WHERE id = $1.
Return `ok({ deleted: true, id: body.id })`.

**Default:** Return `err('INVALID_ACTION', 'action must be one of: create, update, delete, list')`.
