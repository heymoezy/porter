## regenerateMdFiles(templateId: string, trigger: 'star_up' | 'level_milestone' | 'skill_change' | 'equipment_change' | 'full'): Promise&lt;void&gt;

Reads current state from DB, generates markdown content, writes to persona directory.

**Step 1: Fetch agent data**
```sql
SELECT
  at.id, at.name, at.role, at.description,
  at.shell, at.shell_icon, at.shell_color,
  at.intelligence, at.supports, at.equipment_slots,
  at.passive_tree, at.specialties, at.level, at.xp,
  at.star_level, at.rarity, at.elo_rating,
  ars.quality, ars.speed, ars.efficiency, ars.reliability, ars.combo,
  ars.dispatch_count, ars.battle_count
FROM agent_templates at
LEFT JOIN agent_rpg_stats ars ON ars.template_id = at.id
WHERE at.id = $1
```

**Step 2: Fetch active skills**
```sql
SELECT skill_name, success_rate_30d, total_uses
FROM template_skills
WHERE template_id = $1
ORDER BY total_uses DESC NULLS LAST
```

**Step 3: Resolve persona directory**
```typescript
const personaDir = path.join(config.personasDir, templateId);
if (!fs.existsSync(personaDir)) fs.mkdirSync(personaDir, { recursive: true });
```

**Step 4: Generate and write files based on trigger**

`trigger === 'star_up' || trigger === 'full'` → write SOUL.md:
```markdown
# SOUL — {agent.name}

**Shell:** {agent.shell} {agent.shell_icon}
**Rarity:** {agent.rarity} | **Stars:** {'★'.repeat(agent.star_level)}
**Born from:** {agent.description}

## Identity

{agent.role}

## Core Drive

This agent was forged from {agent.dispatch_count} real dispatches.
Every stat reflects actual performance — no designer fiat.

## Specialties

{specialties.map(s => `- ${s.label}`).join('\n') || '- None yet — earn through battles and dispatches'}

_Last regenerated: {new Date().toISOString()}_
```

`trigger === 'level_milestone' || trigger === 'full'` → write IDENTITY.md (only at level multiples of 10, unless trigger==='full'):
```markdown
# IDENTITY — {agent.name}

**Level:** {agent.level} | **XP:** {agent.xp}
**Elo:** {agent.elo_rating}

## Stats

| Stat | Value |
|------|-------|
| Quality (QTY) | {quality.toFixed(1)} |
| Speed (SPD) | {speed.toFixed(1)} |
| Efficiency (EFF) | {efficiency.toFixed(1)} |
| Reliability (REL) | {reliability.toFixed(1)} |
| Synergy (COMBO) | {combo.toFixed(1)} |

## Battle Record

{battle_count} battles | {dispatch_count} dispatches

_Last regenerated: {new Date().toISOString()}_
```

`trigger === 'skill_change' || trigger === 'full'` → write SKILLS.md:
```markdown
# SKILLS — {agent.name}

## Active Skills ({skills.length})

{skills.map(s => `### ${s.skill_name}
- Success rate (30d): ${s.success_rate_30d?.toFixed(1) ?? 'N/A'}%
- Total uses: ${s.total_uses ?? 0}`).join('\n\n') || '_(no skills equipped)_'}

## Supports

{JSON.parse(agent.supports || '[]').map((s: {id: string, target_skill: string, measured_impact: string}) =>
  `- **${s.id}** → ${s.target_skill}: ${s.measured_impact}`).join('\n') || '_(no supports equipped)_'}

_Last regenerated: {new Date().toISOString()}_
```

`trigger === 'equipment_change' || trigger === 'full'` → write TOOLS.md:
```markdown
# TOOLS — {agent.name}

## Equipment Slots

{JSON.parse(agent.equipment_slots || '[]').map((slot: {name: string, tool: string}) =>
  `- **${slot.name}:** ${slot.tool}`).join('\n') || '_(no equipment)_'}

## Passive Tree

{JSON.parse(agent.passive_tree || '[]')
  .filter((n: {unlocked: boolean}) => n.unlocked)
  .map((n: {node_id: string, active: boolean}) => `- ${n.node_id}${n.active ? ' [ACTIVE]' : ''}`)
  .join('\n') || '_(no passive nodes unlocked)_'}

_Last regenerated: {new Date().toISOString()}_
```

**Route 1: GET /api/admin/agents/:id/rpg-stats**
```typescript
fastify.get('/:id/rpg-stats', async (req) => {
  const { id } = req.params as { id: string };
  try {
    const stats = await getRpgStats(id);
    if (!stats) return ok({ stats: null, message: 'No RPG stats yet — agent has no dispatches' });
    return ok({ stats });
  } catch (e) {
    return err('INTERNAL_ERROR', 'Failed to fetch RPG stats');
  }
});
```

**Route 2: POST /api/admin/agents/:id/rpg-recalculate**
```typescript
fastify.post('/:id/rpg-recalculate', async (req) => {
  const { id } = req.params as { id: string };
  try {
    const stats = await recalculateStats(id);
    return ok({ stats, recalculated: true });
  } catch (e) {
    return err('INTERNAL_ERROR', 'Recalculation failed');
  }
});
```

Note: `id` here is a template_id (agent_templates.id). The route parameter name matches the
existing pattern in this file (`/:id/` for template-level operations).
