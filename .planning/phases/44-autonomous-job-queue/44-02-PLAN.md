   - Import: `import { FastifyInstance } from 'fastify';`, `import { pool } from '../../../db/client.js';`, `import { ok } from '../../../lib/admin-envelope.js';`
   - Export default async function that registers 4 routes:

   **GET '/' — list all jobs with filtering:**
   - Parse query params: `status` (string|null), `source` (string|null), `trigger_type` (string|null), `limit` (number, default 50, max 200), `offset` (number, default 0)
   - Build dynamic WHERE conditions array + params array (same pattern as bridge.ts:693)
   - Count query: `SELECT COUNT(*) AS total FROM agent_jobs aj ${where}`
   - Data query:
     ```sql
     SELECT aj.*,
            p.name AS assigned_agent_name,
            CASE WHEN aj.completed_at IS NOT NULL AND aj.started_at IS NOT NULL
                 THEN ROUND((aj.completed_at - aj.started_at) * 1000)::int
            END AS duration_ms,
            LEFT(aj.result, 200) AS result_preview
     FROM agent_jobs aj
     LEFT JOIN personas p ON p.id = aj.agent_id AND aj.agent_id != 'system'
     ${where}
     ORDER BY aj.created_at DESC
     LIMIT $N OFFSET $M
     ```
   - Return: `reply.send(ok({ jobs: rows, total: parseInt(countResult.rows[0]?.total) || 0, limit, offset }))`

   **GET '/queue' — shortcut for pending + running:**
   - Fixed WHERE: `aj.status IN ('pending', 'running')`
   - Same SELECT as above but ordered by `aj.scheduled_for ASC` (next-to-run first)
   - Limit 100, no offset

   **GET '/history' — completed + failed:**
   - Fixed WHERE: `aj.status IN ('complete', 'failed')`
   - Same SELECT as above, ordered by `aj.completed_at DESC NULLS LAST`
   - Parse `limit` (default 50, max 200) and `offset` (default 0) from query

   **GET '/:jobId' — single job detail:**
   - Params: `jobId` from URL
   - Query:
     ```sql
     SELECT aj.*,
            p.name AS assigned_agent_name,
            CASE WHEN aj.completed_at IS NOT NULL AND aj.started_at IS NOT NULL
                 THEN ROUND((aj.completed_at - aj.started_at) * 1000)::int
            END AS duration_ms
     FROM agent_jobs aj
     LEFT JOIN personas p ON p.id = aj.agent_id AND aj.agent_id != 'system'
     WHERE aj.id = $1
     ```
   - Return 404 with error if not found: `reply.code(404).send({ ok: false, error: { code: 'NOT_FOUND', message: 'Job not found' } })`
   - Return full job object (no result_preview truncation — full result field)
