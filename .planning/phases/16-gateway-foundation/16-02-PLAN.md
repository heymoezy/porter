**Main exported function `detectAndUpsertGateways(pool: pg.Pool): Promise<void>`:**

1. Call `bootstrapEnvGateways(pool)` first to handle OLLAMA_URL and OPENCLAW_URL/TOKEN env vars
2. Loop through `CLI_BINARIES`, for each:
   - `const binaryPath = await which(cli.binary).catch(() => null);`
   - If found: call `upsertGateway(pool, { type, name, source: 'auto_detected', status: 'active', metadata: { binary_path: binaryPath }, capabilities })` and log `[bridge] checkmark ${cli.binary} detected at ${binaryPath}`
   - If not found: call `markStale(pool, cli.type, 'auto_detected')` to set status='stale' for that type+source combo (only if row exists)
3. Log summary: `[bridge] Gateway detection complete`

**Helper function `bootstrapEnvGateways(pool: pg.Pool): Promise<void>`:**

1. **Ollama:** Always bootstrap since `config.ollamaUrl` has a default. Call `upsertGateway(pool, { type: 'ollama', name: 'Ollama (local)', url: config.ollamaUrl, authMethod: 'none', source: 'env_bootstrap', status: 'active', capabilities: ['chat', 'code', 'streaming'], metadata: {} })`

2. **OpenClaw:** Only if `config.openclawToken` is non-empty:
   - Upsert gateway row: `{ type: 'openclaw', name: 'OpenClaw', url: config.openclawUrl, authMethod: 'bearer_token', source: 'env_bootstrap', status: 'active', capabilities: ['chat', 'code', 'streaming'], metadata: {} }`
   - If `validatePorterSecret()` returns true, encrypt the token and upsert a credential row:
     ```typescript
     const encrypted = encryptCredential(config.openclawToken);
     const masked = '****...' + config.openclawToken.slice(-4);
     ```
     Then INSERT INTO gateway_credentials with ON CONFLICT update
   - If PORTER_SECRET missing, log warning: `[bridge] PORTER_SECRET not set, skipping OpenClaw credential encryption`

**Helper function `upsertGateway(pool, params)`:**
Uses raw SQL (not Drizzle) since this runs at boot before Drizzle is fully set up:
```sql
INSERT INTO gateways (id, type, name, url, auth_method, status, source, priority, capabilities, metadata, enabled, created_at, updated_at)
VALUES ($1, $2, $3, $4, $5, $6, $7, 10, $8, $9, 1, EXTRACT(EPOCH FROM NOW()), EXTRACT(EPOCH FROM NOW()))
ON CONFLICT (type, source) WHERE source IN ('auto_detected', 'env_bootstrap')
DO UPDATE SET
  name = EXCLUDED.name,
  url = EXCLUDED.url,
  auth_method = EXCLUDED.auth_method,
  status = EXCLUDED.status,
  capabilities = EXCLUDED.capabilities,
  metadata = EXCLUDED.metadata,
  updated_at = EXTRACT(EPOCH FROM NOW())
```
Generate `id` with `crypto.randomUUID()`.

**Helper function `upsertCredential(pool, gatewayType, encryptedValue, maskedDisplay)`:**
```sql
INSERT INTO gateway_credentials (id, gateway_id, label, encrypted_value, masked_display, created_at)
SELECT $1, g.id, 'primary', $2, $3, EXTRACT(EPOCH FROM NOW())
FROM gateways g WHERE g.type = $4 AND g.source = 'env_bootstrap'
ON CONFLICT (id) DO UPDATE SET
  encrypted_value = EXCLUDED.encrypted_value,
  masked_display = EXCLUDED.masked_display
```
Use a deterministic credential ID: `crypto.createHash('sha256').update(gatewayType + ':env_bootstrap:primary').digest('hex').slice(0, 36)` so re-runs upsert the same row.

**Helper function `markStale(pool, type, source)`:**
```sql
UPDATE gateways SET status = 'stale', updated_at = EXTRACT(EPOCH FROM NOW())
WHERE type = $1 AND source = $2 AND status != 'stale'
```

Wrap the entire `detectAndUpsertGateways` body in try/catch — log errors but do NOT throw (startup must not crash if detection fails):

try { ... } catch (err) {
  console.error('[bridge] Gateway detection failed:', err instanceof Error ? err.message : err);
}
