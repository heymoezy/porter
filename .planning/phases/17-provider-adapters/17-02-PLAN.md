<!-- Wire formats from research — verified from actual CLI runs -->

**Codex CLI JSONL format:**
- Spawn: `codex exec --json --ephemeral --skip-git-repo-check "prompt text"` (prompt as positional arg)
- Token events: `{"type":"item.completed","item":{"type":"message","content":[{"type":"output_text","text":"Hello!"}]}}`
- Error: `{"type":"error","message":"..."}` or `{"type":"turn.failed","error":{"message":"..."}}`
- Timeout: 120s (Codex is slower)

**Claude CLI stream-json format:**
- Spawn: `claude -p --output-format stream-json --verbose --include-partial-messages --no-session-persistence`
- Prompt delivery: write to stdin, then close stdin
- Streaming tokens: `{"type":"stream_event","event":{"type":"content_block_delta","index":0,"delta":{"type":"text_delta","text":"Hey"}}}`
- Final usage: `{"type":"result","subtype":"success","result":"...","usage":{"input_tokens":N,"output_tokens":N}}`
- Model detection: `{"type":"system","subtype":"init","model":"claude-sonnet-4-6"}`

**Gemini CLI stream-json format:**
- Spawn: `gemini -p "prompt text" --output-format stream-json --yolo`
- Streaming: `{"type":"message","role":"assistant","content":"Hello","delta":true}`
- Final: `{"type":"result","status":"success","stats":{"input_tokens":N,"output_tokens":N}}`
- Model: `{"type":"init","model":"auto-gemini-3"}`

**health()**: `const start = Date.now()`. Spawn `codex --version` (or `codex -V`) with a 5s timeout. If process exits with code 0, return `{ healthy: true, latencyMs }`. If it fails or times out, return `{ healthy: false, error }`. Drain stderr with `child.stderr.resume()`.

**dispatch(req)**: Extract last user message: `const prompt = req.messages.filter(m => m.role === 'user').at(-1)?.content ?? ''`. Spawn: `codex exec --json --ephemeral --skip-git-repo-check` with prompt as positional argument (append to args array). Set stdio to `['pipe', 'pipe', 'pipe']`. Pass `process.env` as env. Close stdin immediately with `child.stdin.end()`. Set timeout to 120_000ms (Codex is slower). Drain stderr with `child.stderr.resume()`.

Use `createInterface({ input: child.stdout!, terminal: false })` and iterate with `for await (const line of rl)`. Collect response text from JSONL:
- Skip empty lines
- Parse each line as JSON
- If `event.type === 'error'`: throw `new Error('Codex error: ' + event.message)`
- If `event.type === 'turn.failed'`: throw `new Error('Codex error: ' + (event.error as any)?.message)`
- If `event.type === 'item.completed' && (event.item as any)?.type === 'message'`: iterate `(event.item as any).content` array, collect all entries where `c.type === 'output_text'` into response text

After readline completes, clear timeout, wait for child exit. Return `BridgeDispatchResult`:
- `response`: collected text
- `model`: `req.model ?? 'gpt-5.4'`
- `latencyMs`: `Date.now() - start`
- `cached`: `false`
- `tokensUsed`: undefined (Codex does not report token counts in exec mode)

**stream(req, signal)**: Async generator. Same spawn pattern as dispatch. On `signal` abort, call `child.kill('SIGTERM')` and return. For each JSONL line, same parsing as dispatch but `yield` each `output_text` content chunk instead of collecting. Handle `error` and `turn.failed` events by throwing. On timeout (120s), kill with SIGTERM.

Add an AbortSignal listener: `const onAbort = () => { child.kill('SIGTERM'); }; signal.addEventListener('abort', onAbort, { once: true });` Remove listener in finally block.

**listModels()**: Return `['gpt-5.4']` (static — from Codex config.toml default).

CRITICAL: Prompt is a positional argument to `codex exec` (NOT stdin). Always use `--skip-git-repo-check`. Always `child.stderr.resume()` to prevent deadlock. Always clear timeout in finally.

```
gemini -p "prompt text" --output-format stream-json --yolo
```
Prompt is passed as the value after `-p` flag (positional argument to -p, NOT stdin). Close stdin immediately. Timeout: 60_000ms. Drain stderr.

Parse JSONL lines:
- On `init` event (type === 'init'): capture `model` field
- On `message` event (type === 'message', role === 'assistant', delta absent or false): this is the final complete message. Collect `content`.
- On `message` event with `delta === true`: this is a streaming partial — also collect content (for dispatch, we want the final one)
- On `result` event (type === 'result'): extract `stats.input_tokens`, `stats.output_tokens`
- On `error` event (type === 'error'): throw with error message

Use the last complete message (where delta is absent/false) as the response text. If only delta messages exist, concatenate all delta content values.

Return `BridgeDispatchResult`:
- `response`: final message content
- `model`: captured model or `req.model ?? 'gemini-2.5-flash'`
- `inputTokens`: from result.stats
- `outputTokens`: from result.stats
- `tokensUsed`: input + output
- `latencyMs`: elapsed
- `cached`: false

**stream(req, signal)**: Async generator. Same spawn as dispatch. Add abort listener on signal. For each JSONL line:
- If `event.type === 'message' && event.role === 'assistant' && event.content`: yield `event.content`. This works for both delta=true (streaming partials) and final messages.
- Skip `init`, `result`, `tool_use`, `tool_result` events.
- On `error` event: throw.
- On signal abort, kill child and return.

Timeout: 60s with SIGTERM. Always clear timeout in finally. Always drain stderr.

**listModels()**: Return `['gemini-2.5-pro', 'gemini-2.5-flash', 'auto-gemini-3']` (static — from Gemini CLI init event discovery).

CRITICAL: Prompt goes as argument to `-p` flag (NOT stdin). Must use `--yolo` flag to suppress confirmation prompts. Must drain stderr (keychain warnings from libsecret). Accept `auto-*` prefixed model names as valid.
