## Decisions Made

- **Mixed Drizzle + raw pool in auth.ts:** auth.ts uses Drizzle ORM for session/user lookups (high-level, entity-based) but raw pool.query for the collaborator check query (already written as raw SQL in the plan). This mixed pattern is intentional and correct — Drizzle is appropriate for ORM-style entity operations, raw pool for complex SQL.
- **Single large migration commit:** All 12 service files were converted in one commit (e53ae59) rather than per-file commits. This was the approach taken by the prior session and is acceptable for a batch migration with no inter-service dependencies.
