# Contract Reviewer — Example Output Shapes

Use these as patterns for practical contract-review outputs.

## Example 1 — MSA review

**Input:**
Review this vendor MSA and tell me what we should push back on.

**Good output shape:**
- contract summary and party posture
- key commercial terms
- must-fix clauses with reasons
- negotiable clauses with fallback positions
- counsel escalation items

## Example 2 — NDA review

**Input:**
Check whether this NDA is balanced enough for us to sign.

**Good output shape:**
- scope of confidential information
- term and survival analysis
- permitted use/disclosure exceptions
- injunctive relief, residuals, and return/destroy obligations
- balanced vs one-sided conclusion

## Example 3 — SaaS order form + online terms

**Input:**
Summarize the risk in this SaaS contract package.

**Good output shape:**
- package structure and incorporated docs
- subscription, payment, renewal, and SLA points
- data/privacy/security obligations
- liability and indemnity summary
- prioritized negotiation list

## Example 4 — Clause rewrite support

**Input:**
Propose safer language for this termination-for-convenience clause.

**Good output shape:**
- current risk explanation
- drafting objective
- redline direction or fallback text
- tradeoffs of each option
- note on counsel review if jurisdiction matters
