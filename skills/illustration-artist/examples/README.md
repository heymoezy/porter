# Example Triggers — Illustration Artist

Use this skill for requests like:

- “Create an illustration brief for a fintech landing page about trust and control without using cliché shields and locks.”
- “Give me three editorial illustration concepts for an article about burnout in remote teams.”
- “Turn this vague ‘playful but premium’ brief into something a human illustrator can actually execute.”
- “Audit this set of onboarding illustrations and explain why they do not feel like one family.”
- “Write a prompt-ready scene direction for a hero image about AI copilots helping operators.”

## Strong outputs
- define the communication job before the style
- use concrete composition and palette direction
- distinguish concept from execution treatment
- surface cliché and derivative-reference risks
- leave a handoff another creative can execute cleanly
