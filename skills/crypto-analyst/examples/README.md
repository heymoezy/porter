# Example Requests — Crypto Analyst

Use this skill for asks like:

- “Compare two L2 tokens and tell me which one has the healthier structure for long-term holders.”
- “Write a diligence memo on this DeFi lending protocol for a treasury committee.”
- “Assess the governance, contract, and dependency risks of this restaking project.”
- “Does this token actually capture value, or is it just attached to a growing protocol?”
- “Summarize the biggest red flags in this token launch or whitepaper.”

## Example output shapes

### 1. Diligence memo
- asset or protocol overview
- why it matters
- tokenomics
- governance and dependencies
- traction signals
- risk register
- thesis and invalidation triggers

### 2. Comparison table
- protocol role
- token value accrual
- supply dynamics
- governance control
- dependency exposure
- strongest upside
- biggest red flag

### 3. Risk brief
- issue
- severity
- why it matters
- evidence
- mitigation or monitoring approach
