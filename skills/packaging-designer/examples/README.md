# Packaging Designer — Example Output Shapes

Use these as patterns for packaging-focused outputs.

## Example 1 — Packaging audit

**Input:**
Critique our protein powder tub for shelf impact and trust.

**Good output shape:**
| Area | Issue | Why it matters | Recommendation |
|---|---|---|---|
| Front panel | product type reads smaller than brand mark | category recognition slows | enlarge “whey protein isolate” and reduce logo dominance |
| Variant cue | flavor is a small badge only | line is hard to shop | dedicate a stronger flavor band + color system |
| Back panel | benefits, directions, and warnings compete | readability and trust drop | regroup into blocks with clearer hierarchy |
| Finish | matte black everywhere | premium look but low information contrast | add high-contrast text zones |

Then add:
- quick wins vs structural redesigns
- production cautions
- premium vs mass-market positioning notes

## Example 2 — New packaging direction

**Input:**
Design packaging direction for a premium candle brand sold online and in boutiques.

**Good output shape:**
- buyer context and channel behavior
- structural format recommendation
- front-panel hierarchy
- scent / SKU system
- material and finish direction
- unboxing notes
- likely production risks

## Example 3 — SKU architecture

**Input:**
We need a system for 12 skincare SKUs with multiple strengths.

**Good output shape:**
| Element | Fixed across line | Variable by SKU | Rule |
|---|---|---|---|
| Brand mark | yes | no | top-left, constant size |
| Product family band | yes | no | fixed position and shape |
| Color | no | yes | hue by routine step |
| Strength indicator | no | yes | numeric badge with one format |
| Icon | style fixed | symbol changes | one icon family only |

Then add:
- how the system scales to future launches
- where detail belongs beyond the front panel

## Example 4 — Dieline-aware copy planning

**Input:**
Help us organize a small carton with lots of required information.

**Good output shape:**
- Front: product, key promise, size
- Side A: directions + ingredients
- Side B: warnings + barcode + contact information
- Back: proof points + short brand story
- Top/bottom: lot code, recycling icons, utility text
- Notes on minimum type size, whitespace, and quiet zones
