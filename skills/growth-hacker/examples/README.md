# Growth Hacker — Example Output Shapes

Use these as patterns for strong growth deliverables.

## Example 1 — Activation bottleneck diagnosis

**Input:**
Signups are rising, but almost nobody reaches first value.

**Good output shape:**
- bottleneck hypothesis
- affected segments
- likely causes ranked by confidence
- top experiments to run next
- primary metric, guardrails, and instrumentation needs

## Example 2 — Retention recovery plan

**Input:**
Week-4 retention dropped hard after our onboarding redesign. What should we test?

**Good output shape:**
- likely failure modes
- retention or habit hypotheses
- experiment backlog ranked by learning value
- leading indicators and guardrails
- dependencies or measurement caveats

## Example 3 — Referral loop review

**Input:**
Do we have a real referral loop or just invites that go nowhere?

**Good output shape:**
- current loop assessment
- friction points in share, acceptance, activation, and reward path
- redesigned loop ideas
- test sequencing plan
- success metrics and abuse / trust guardrails

## Example 4 — Test readout

**Input:**
Interpret this A/B test result and tell us what to do next.

**Good output shape:**
- what changed and for which segment
- whether the lift is likely meaningful
- confounders or data-quality caveats
- keep / iterate / stop recommendation
- next experiment or instrumentation fix
