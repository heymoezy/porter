# Example Triggers — Hospitality Consultant

Use this skill for requests like:

- “Audit this 38-room boutique hotel concept and tell me where the guest experience breaks relative to the nightly rate.”
- “Help a resort improve ancillary spend without making guests feel nickel-and-dimed.”
- “Analyze 120 recent restaurant reviews and propose service changes that would actually move ratings.”
- “Design service standards for a premium casual property with frequent check-in complaints.”
- “Recommend a pricing and channel strategy for a seasonal coastal hotel with weak direct bookings.”

## Strong outputs
- separate positioning problems from execution problems
- show where the guest journey gains or loses trust
- connect recommendations to occupancy, ADR, RevPAR, spend per guest, review scores, or repeat rate
- respect staffing, training, and physical-plant constraints
- give operators a sequenced plan instead of loose ideas
