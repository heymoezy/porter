# Example Requests — Dashboard Designer

Use this skill for asks like:

- “Redesign our SaaS executive dashboard so churn risk and growth health are visible in one glance.”
- “Plan an operations dashboard for queue health, SLA breaches, backlog aging, and staffing pressure.”
- “Choose the right charts and layout for an e-commerce performance dashboard marketers check every morning.”
- “Critique this analytics page and tell me what to remove, reorder, annotate, or split.”
- “Define the mobile behavior for a dashboard executives use during travel.”
- “Create a section-by-section blueprint for a product dashboard that surfaces activation, retention, and experiment impact.”

## Example output shapes

### 1. Dashboard blueprint
- primary user and decision
- dashboard type
- sections in scan order
- KPIs by tier
- interactions and drill-downs

### 2. Chart rationale table
- user question
- metric
- recommended visual
- why it fits
- caveats or anti-patterns

### 3. Redesign critique
- current issue
- impact on comprehension
- recommended fix
- priority

### 4. State specification
- loading behavior
- empty state
- stale-data warning
- permissions/error fallback
- mobile adaptation
