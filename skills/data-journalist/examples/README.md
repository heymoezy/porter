# Example Requests — Data Journalist

Use this skill for asks like:

- “Turn this city housing dataset into three credible story angles and rank them by public-interest value.”
- “Stress-test whether this headline about startup layoffs is actually supported by the data.”
- “Write a findings brief with the trend, the comparison baseline, and the caveats.”
- “Create a chart-led article outline for a public report on wage growth by region.”
- “Draft a methodology note and source ledger for this data-backed explainer.”
- “Show what additional reporting would be needed before publishing this claim.”

## Example output shapes

### 1. Story-angle memo
- angle
- why it matters now
- supporting evidence
- caveats
- reporting gaps
- headline directions

### 2. Findings brief
- main finding
- key supporting numbers
- baseline/comparison
- method summary
- uncertainty and caveats
- implications

### 3. Claim ledger
- claim
- source/data support
- confidence level
- caveat
- additional verification needed

### 4. Chart sequence
- chart purpose
- key message
- annotation/context needed
- reader risk of misinterpretation
