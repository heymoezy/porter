# Example Triggers — Icon Designer

Use this skill for requests like:

- “Design a 20px outline icon family for billing, invoices, payouts, taxes, and subscriptions.”
- “Audit this mixed icon set and explain why it feels inconsistent.”
- “Write implementation specs for a rounded, filled icon system that needs to match iOS-style softness.”
- “Give me better metaphors for notifications, automations, workflows, and integrations.”
- “Turn these product nouns into a coherent icon brief for a designer or image model.”

## Strong outputs
- define the family rules before listing icons
- explain metaphor choice and likely collisions
- account for target size and context
- use concrete geometry and style language
- leave a repeatable expansion path for future icons
