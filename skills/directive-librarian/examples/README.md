# Directive Librarian — Example Output Shapes

Use these shapes to keep directive work precise.

## Example 1 — Promote a lesson into a directive

**Input:**
We keep deploying without rollback notes. Turn this lesson into a directive.

**Good output shape:**
- source evidence
- normalized directive text
- scope and trigger
- status recommendation
- owner / approver
- related directives and cleanup actions

## Example 2 — Resolve overlap

**Input:**
Two directives describe permission approvals differently. Clean this up.

**Good output shape:**
- directives compared
- exact overlap or conflict
- whether one is a scoped exception
- merge / supersede / dispute recommendation
- updated status guidance

## Example 3 — Catalog hygiene audit

**Input:**
Audit this directive set for duplicates, stale rules, and weak provenance.

**Good output shape:**
- duplicate clusters
- disputed or deprecated candidates
- missing provenance cases
- retrieval or naming issues
- cleanup priority list

## Example 4 — Status classification

**Input:**
This instruction came from a Slack thread. Is it an active directive?

**Good output shape:**
- provenance assessment
- scope assessment
- authority / approval assessment
- recommended status
- evidence or review still required
