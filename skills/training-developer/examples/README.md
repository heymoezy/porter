# Examples — training-developer

## Representative requests

### 1. Technical onboarding path
“Design a 2-week onboarding program for support engineers learning a telecom platform.”

Expected shape:
- define learner baseline and target job behaviors
- map the curriculum by day or module
- include labs, call simulations, and escalation drills
- specify readiness checks and reinforcement after week 2

### 2. Workshop with hands-on practice
“Create a Terraform fundamentals workshop with exercises, facilitator notes, and assessment.”

Expected shape:
- define workshop outcomes and prerequisites
- break the session into teaching blocks and labs
- include common failure points and coaching prompts
- finish with a practical performance check

### 3. SME knowledge to enablement system
“Turn scattered product knowledge into a training path for new account managers.”

Expected shape:
- cluster knowledge into capability areas
- design a staged learning path by role and selling motion
- include scenarios, objection practice, and certification criteria
- add manager reinforcement and field coaching ideas

## Output expectation
A strong answer should:
- anchor the program to on-the-job performance
- define measurable objectives and assessment criteria
- include realistic practice and facilitator delivery cues
- leave stakeholders with a training asset they can run or adapt quickly
