# Strategic Planner — Example Output Shapes

Use these patterns to keep strategic work decision-ready instead of abstract.

## Example 1 — Growth-path choice

**Input:**
Assess whether a B2B SaaS company should expand upmarket, deepen self-serve, or focus on retention for the next 12 months.

**Good output shape:**
- one-sentence decision statement
- current-state diagnosis with the few facts that matter most
- 3 option writeups with upside, downside, capability load, and timing
- recommended path with why it fits now
- leading indicators and kill criteria

## Example 2 — Portfolio prioritization

**Input:**
We have too many strategic initiatives. Build a prioritization recommendation for leadership.

**Good output shape:**
- strategic objective and hard constraints
- initiative clustering by outcome and dependency
- what to stop, defer, combine, or fund harder
- recommended sequence across now / next / later
- political / change-management risks to manage explicitly

## Example 3 — Build / buy / partner memo

**Input:**
Write a decision memo comparing build, buy, and partner options for a new capability.

**Good output shape:**
- decision frame and evaluation criteria
- option comparison table with differences that matter
- hidden costs and failure modes for each path
- recommendation with assumptions
- trigger points that would change the recommendation later

## Example 4 — Market-entry sequencing

**Input:**
Create a practical market-entry strategy for expanding into one new region without breaking the core business.

**Good output shape:**
- market-entry thesis and non-negotiables
- entry options with resource and risk patterns
- phased sequence with evidence gates
- capability gaps and ownership implications
- explicit “not now” items to keep focus intact
