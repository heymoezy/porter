# Sustainability Advisor — Example Deliverable Shapes

Use these as patterns for credible, action-oriented outputs.

## Example 1 — Practical decarbonization roadmap

**Input:**
Build a realistic decarbonization plan for a mid-sized company with patchy Scope 3 data.

**Good output shape:**
- operating context and major hotspot assumptions
- what is known, estimated, and missing
- ranked reduction levers by impact, feasibility, and owner
- reporting and data-improvement steps in parallel
- milestones, caveats, and proof requirements

## Example 2 — Claim credibility review

**Input:**
Assess whether the company’s current sustainability claims are defensible.

**Good output shape:**
- claims inventory and implied promises
- evidence and boundary gaps by claim
- high-risk wording to remove or qualify
- what substantiation would be needed to keep stronger language
- recommended safer claim architecture

## Example 3 — Reporting-readiness memo

**Input:**
Evaluate whether the business is ready to report Scope 1, 2, and material Scope 3 categories.

**Good output shape:**
- current boundary/methodology state
- control and data gaps by scope/category
- estimation quality and traceability issues
- near-term actions for reporting readiness
- what should not be claimed yet

## Example 4 — Supply-chain sustainability priorities

**Input:**
Prioritize sustainability actions across suppliers, logistics, packaging, and facilities.

**Good output shape:**
- hotspot map across the value chain
- leverage assessment by function
- options ranked by impact, feasibility, and timeline
- dependencies on procurement, operations, and finance
- leading indicators to track progress

## Example 5 — Target-setting support

**Input:**
Stress-test a proposed emissions-reduction target before leadership announces it.

**Good output shape:**
- baseline and scope assumptions
- realism check on reduction pathway
- operational levers required to deliver
- offset reliance and credibility risks
- recommendation: announce, revise, or defer
