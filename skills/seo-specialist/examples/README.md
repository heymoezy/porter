# SEO Specialist — Example Output Shapes

Use these patterns for concise, implementation-ready SEO work.

## 1) Page optimization brief

**Input:**
Optimize this landing page for organic search.

**Good output shape:**
- target intent and probable SERP expectation
- title, H1, and structure recommendations
- content clarity / gap notes
- internal-linking improvements
- technical blockers
- priority order

## 2) Topic cluster planning

**Input:**
What should we publish to win this topic area?

**Good output shape:**
- grouped intents and page roles
- pillar vs supporting content map
- cannibalization risks
- internal-link architecture
- priority sequence by opportunity and effort

## 3) Technical SEO audit

**Input:**
Audit this site for crawl and indexing issues.

**Good output shape:**
- highest-impact discovery/indexability blockers
- canonical, redirect, render, or duplicate-content issues
- likely effect on visibility
- remediation order
- caveats or missing data

## 4) CTR improvement

**Input:**
Our pages rank but clicks are weak. Help.

**Good output shape:**
- likely snippet / intent mismatch causes
- title and meta description revisions
- differentiation or proof cues
- structured-data opportunities if relevant
- measurement plan for changes

## 5) Underperforming article diagnosis

**Input:**
Why is this article struggling in organic search?

**Good output shape:**
- likely intent mismatch or SERP-shape mismatch
- topic coverage gaps
- structural and clarity issues
- internal-linking / freshness recommendations
- highest-confidence fixes first
