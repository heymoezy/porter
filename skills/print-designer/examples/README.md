# Print Designer — Example Output Shapes

## Example 1 — Prepress review table

**Input:**
Review this tri-fold brochure before we send it to print.

**Good output shape:**
| Issue | Severity | Why it matters | Fix |
|---|---|---|---|
| Background image stops at trim | High | white slivers can appear after cutting | extend artwork through bleed |
| Contact details sit inside fold zone | High | readability drops and fold can damage legibility | move into safe panel area |
| Body text reversed out at small size | Medium | ink spread can fill counters and reduce clarity | increase size/weight or use dark-on-light |
| Hero image placed below effective resolution target | Medium | softness visible in print | replace or reduce print size |

Then add:
- export assumptions
- proofing notes
- unresolved printer questions

## Example 2 — Poster guidance for distance viewing

**Input:**
Help structure a conference poster for hallway traffic.

**Good output shape:**
- viewing-distance assumption
- primary message hierarchy
- headline / subhead / body sizing guidance
- contrast and non-backlit readability notes
- recommended print specs and material questions

## Example 3 — Booklet layout risk memo

**Input:**
What should I check before exporting a saddle-stitched event booklet?

**Good output shape:**
- finished size and page-count assumptions
- creep / gutter / crossover image considerations
- margin and page-number placement risks
- image and black-build checks
- export / proof checklist

## Example 4 — Print handoff summary

**Input:**
Prepare a handoff note for our A5 event flyer.

**Good output shape:**
- finished dimensions
- bleed and safe-zone assumptions
- color mode and spot-color notes if any
- font / image readiness status
- preferred PDF export standard or printer confirmation request
- sign-off items before release
