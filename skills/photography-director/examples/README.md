# Photography Director — Example Output Shapes

Use these as response patterns, not rigid templates.

## Example 1 — Product shoot shot list

**Input:**
Plan a shoot for a premium skincare brand launching two serums.

**Good output shape:**
| Priority | Shot | Purpose | Notes |
|---|---|---|---|
| Must | straight-on hero bottle | PDP / retail listing | label legible, clean gradient background, soft specular control |
| Must | texture dispense macro | prove formula quality | controlled droplet shape, crisp viscosity detail |
| Must | hand-in-use routine shot | show context and scale | natural skin finish, morning-bathroom light feel |
| Should | paired product composition | brand / social | premium symmetry, negative space for copy |
| Nice | packaging + secondary prop vignette | campaign variation | restrained prop world, no clutter |

Then add:
- lighting setup logic
- prop / styling notes
- crop requirements by channel

## Example 2 — Brand photography direction memo

**Input:**
Develop a photography direction for a modern law-firm website.

**Good output shape:**
- image objective
- emotional register
- casting / subject direction rules
- lighting and composition principles
- location choices
- cliché avoidance list
- homepage vs practice-page asset split

## Example 3 — Image critique

**Input:**
Review our conference photos and tell us what to reshoot next time.

**Good output shape:**
- what worked
- repeated misses in coverage
- technical issues hurting usability
- weak moments / backgrounds / crops
- revised capture priorities for the next event

## Example 4 — One shoot, two channels

**Input:**
We need one photography concept that works for ecommerce and social.

**Good output shape:**
- shared visual system
- asset list by channel
- what must stay consistent
- where to allow more energy or variation for social
- schedule plan that captures both efficiently
