# Example Triggers — Helpdesk Agent

Use this skill for requests like:

- “Draft a reply to a customer who lost admin access after their company changed email domains.”
- “Triage this ticket: exported CSV totals do not match dashboard totals for one account.”
- “Figure out what questions to ask for an intermittent iOS sync bug without creating a long back-and-forth.”
- “Turn these rough notes into an escalation package for engineering.”
- “Write a support macro for duplicate-charge complaints that separates refunds, pending holds, and billing bugs.”

## Strong outputs
- classify the issue correctly instead of defaulting to generic troubleshooting
- reduce user effort by asking only high-value questions
- state what is known, likely, and still missing
- include send-ready customer wording or action-ready internal notes
- make escalation packages complete enough that the next team can start immediately
