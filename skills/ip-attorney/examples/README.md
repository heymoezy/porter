# Example requests

- Assess launch risk for a new product name that overlaps with several adjacent trademarks in overlapping channels.
- Summarize the IP diligence risks in an acquisition where core software was built by contractors and uses multiple OSS libraries.
- Outline patent-landscape and invention-capture considerations for an AI document automation workflow.
- Review whether a content licensing deal leaves enough rights for derivative works, sublicensing, and international distribution.

## Strong deliverable patterns
- Start with the business objective, jurisdiction, and right type.
- Separate known facts from assumptions and open questions.
- Use a risk matrix or option table when the user must choose between proceed, modify, license, or escalate.
- State what screening-level analysis can and cannot resolve.
- End with the fastest sensible next step to reduce legal uncertainty.
