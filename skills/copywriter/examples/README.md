# Copywriter — Example Output Shapes

Use these as patterns for persuasion-heavy tasks.

## Example 1 — Landing page hero

**Input:**
Rewrite our homepage hero for a workflow automation tool targeting operations teams.

**Good output shape:**
- primary headline
- supporting subhead
- 3 proof/benefit bullets
- primary CTA and secondary CTA
- 2 alternate headline angles

## Example 2 — Paid ad variants

**Input:**
Create LinkedIn ad copy for a B2B analytics product launch.

**Good output shape:**
- 3 angle-based variants
- each variant includes hook, body, CTA
- short note on when to test each angle

## Example 3 — Objection-handling copy

**Input:**
Draft website copy addressing why buyers think migration will be too hard.

**Good output shape:**
- objection statement
- reframed response
- proof or mechanism detail
- CTA or reassurance line

## Example 4 — Email nurture sequence

**Input:**
Write a 4-email nurture sequence for trial users who have not activated key features.

**Good output shape:**
- goal for each email
- subject line options
- concise body copy
- CTA per email
- sequencing logic tied to friction reduction
