# Course Creator — Example Output Shapes

Use these as patterns for instructional-design tasks.

## Example 1 — Cohort course blueprint

**Input:**
Design a 6-week cohort course teaching product managers how to use AI for research and planning.

**Good output shape:**
- learner profile and outcomes
- week-by-week module plan
- assignment and feedback structure
- live session agenda pattern
- capstone/project definition

## Example 2 — Employee onboarding program

**Input:**
Create a two-week onboarding curriculum for new support agents.

**Good output shape:**
- role readiness outcomes
- day-by-day learning path
- practice scenarios
- certification or readiness checks
- manager/instructor notes

## Example 3 — Workshop design

**Input:**
Build a 90-minute workshop on prompt evaluation for non-technical staff.

**Good output shape:**
- session objective
- time-boxed agenda
- mini-lecture sections
- interactive exercise
- debrief questions
- take-home resources

## Example 4 — Self-paced module set

**Input:**
Turn this SME document set into an async learning path.

**Good output shape:**
- module breakdown
- objective per module
- content chunking approach
- quiz or self-check plan
- points where examples, practice, and feedback should appear
