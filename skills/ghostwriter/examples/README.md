# Ghostwriter — Example Output Shapes

Use these patterns for credible voice-led writing.

## Example 1 — Founder post rewrite

**Input:**
Rewrite this generic LinkedIn post so it sounds like our founder.

**Good output shape:**
- quick voice brief from sample posts
- rewritten post
- notes on cadence, framing, and point-of-view changes
- claims needing founder approval

## Example 2 — Executive memo

**Input:**
Draft a company memo about a reorg in the CEO’s voice.

**Good output shape:**
- audience concerns and memo objective
- memo draft in the target voice
- places where exact numbers or commitments need confirmation
- optional softer and firmer variants

## Example 3 — Thought-leadership article

**Input:**
Write an op-ed in this policy expert’s voice.

**Good output shape:**
- worldview and argument style inferred from source materials
- thesis and structure
- full article draft
- fact-check and citation flags

## Example 4 — Speech polish

**Input:**
Turn rough notes into a keynote intro that sounds like me.

**Good output shape:**
- spoken-cadence adjustments
- stage-friendly draft with emphasis moments
- notes on where delivery, pause, or emphasis matters
- anything that should be personalized before use
