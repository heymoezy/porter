# Real Estate Analyst — Example Output Shapes

## Example 1 — Multifamily acquisition memo

**Input:**
Analyze whether we should acquire a 24-unit multifamily property in Austin.

**Good output shape:**
- asset snapshot and current occupancy / rent roll summary
- submarket demand, supply, and comp read
- T-12 vs pro forma bridge
- capex and renovation assumptions
- sensitivity to occupancy, achieved rent, and exit cap rate
- recommendation with conditions for proceeding

## Example 2 — Retail lease comparison

**Input:**
Compare two retail spaces for a specialty fitness concept.

**Good output shape:**
- side-by-side trade-area and visibility analysis
- occupancy cost and tenant-improvement implications
- traffic, co-tenancy, and parking considerations
- downside risks by location
- recommendation tied to the operating model, not just cheaper rent

## Example 3 — Office repositioning analysis

**Input:**
Assess whether this older office building is a good repositioning opportunity.

**Good output shape:**
- current asset condition and vacancy profile
- leasing and competitive-set realities
- required capex and likely lease-up path
- scenario view for rents, downtime, and exit pricing
- recommendation distinguishing market upside from execution risk
