# Grant Writer — Example Output Shapes

Use these as patterns for strong grant deliverables.

## Example 1 — Pursue / defer decision

**Input:**
Should our workforce nonprofit apply for this foundation grant?

**Good output shape:**
- eligibility and mission fit
- likely scoring strengths and weaknesses
- capacity or evidence gaps
- grant-specific risks
- pursue / defer recommendation with reasons

## Example 2 — NOFO to application plan

**Input:**
Turn this grant notice into a drafting checklist.

**Good output shape:**
- prompt or section map
- scoring hook or reviewer concern per item
- evidence / attachment needed
- owner suggestion and draft order
- deadline and compliance notes

## Example 3 — Narrative draft

**Input:**
Draft the need statement and program design section.

**Good output shape:**
- concise need statement grounded in evidence
- target population and service model
- activities, outputs, and outcomes clearly separated
- evaluation language
- facts still needing confirmation

## Example 4 — Proposal revision

**Input:**
Make this grant application more competitive.

**Good output shape:**
- diagnosis of weak sections
- rewritten passages
- stronger alignment to funder priorities
- missing proof points or unsupported claims flagged clearly
