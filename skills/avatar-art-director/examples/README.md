# Avatar Art Director — Example Output Shapes

Use these as patterns for strong avatar-direction deliverables.

## Example 1 — Single assistant avatar direction

**Input:**
Define the avatar direction for a trustworthy but sharp AI operator assistant.

**Good output shape:**
- persona summary and dominant traits
- recommended visual archetype
- silhouette, expression, palette, and accessory guidance
- what to avoid: too cute, too sci-fi, too corporate, too human-like
- thumbnail-readability notes
- brief for the illustrator or generator

## Example 2 — Family of role avatars

**Input:**
We need a set of avatars for researcher, builder, reviewer, and dispatcher roles.

**Good output shape:**
- shared family rules
- role-specific differentiators
- crop/framing consistency
- accent-color or prop logic
- expression range boundaries
- QA notes to prevent the set from drifting apart visually

## Example 3 — Critique of current concepts

**Input:**
Review these six avatar drafts and tell us which one should ship.

**Good output shape:**
- strongest concept and why
- what each draft communicates in one glance
- legibility risks at real size
- unnecessary details to remove
- exact revisions for the chosen concept
- rejection reasons for weak concepts

## Example 4 — Pixel avatar brief

**Input:**
Create direction for a pixel-art avatar representing a tactical planner.

**Good output shape:**
- core role signal
- pixel-appropriate shape and palette rules
- face/expression constraints
- prop usage guidance
- small-grid simplification rules
- examples of on-brief vs off-brief choices

## Example 5 — Repositioning an existing avatar

**Input:**
Our bot avatar feels generic. Make it feel more premium and competent.

**Good output shape:**
- diagnosis of the current generic signals
- new priority traits
- revised visual rules
- changes to silhouette, expression, palette, and detail density
- rollout guidance for updating legacy variants
