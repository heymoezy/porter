# Example Triggers — Infographic Designer

Use this skill for requests like:

- “Turn this dense research summary into a one-page infographic for city officials.”
- “Design a 7-slide carousel explaining our onboarding funnel drop-off.”
- “Recommend the best structure for an infographic comparing three insurance plans.”
- “Convert this operations SOP into a visual process explainer for new hires.”
- “Give me a prompt-ready layout brief for a healthcare outcomes infographic.”

## Strong output traits
- Names the audience, format, and single takeaway up front.
- Chooses a story structure that matches the content instead of forcing a trendy layout.
- States what becomes visual, what stays textual, and what gets removed.
- Uses chart choices and annotation logic that support honest comparison.
- Produces a handoff-ready plan, wireframe, or creative brief instead of vague style advice.

## Anti-patterns
- Treating every infographic like a long poster with equal-weight sections.
- Using decorative icons or illustrations where labels or charts should do the work.
- Overstuffing the asset instead of recommending cuts or a multi-frame sequence.
- Suggesting chart types that obscure ranking, trend, or scale.
