# Community Manager — Example Output Shapes

Use these as patterns for strong community-management deliverables.

## Example 1 — Community strategy reset

**Input:**
Our customer Slack has 12,000 members but feels dead. Most questions go unanswered.

**Good output shape:**
- community purpose and target member behaviors
- diagnosis of response and participation gaps
- member segment view: newcomers, experts, lurkers
- recommended rituals, staffing, and channel changes
- KPI scorecard for response health and retention
- first 30-day action plan

## Example 2 — Moderation playbook

**Input:**
Draft moderation rules and escalation steps for a creator Discord server.

**Good output shape:**
- principles and community norms
- rule categories with plain-language examples
- moderator actions by severity
- escalation matrix for harassment, scams, and legal risk
- warning and removal templates
- review cadence for policy tuning

## Example 3 — Engagement calendar

**Input:**
Plan a month of programming for our SaaS user community.

**Good output shape:**
- monthly engagement objective
- recurring formats by week
- prompts, hosts, and prep needs
- activation tactics for low-participation segments
- success metrics per event type
- post-event follow-up workflow

## Example 4 — New-member onboarding

**Input:**
New members join but rarely post. Design an onboarding flow.

**Good output shape:**
- first-session friction points
- welcome and orientation sequence
- low-risk first contribution prompts
- champion or moderator touches
- milestone messages for week 1 and week 4
- activation metrics and drop-off checks

## Example 5 — Response library

**Input:**
Create standard replies for spam, repeat complaints, and unanswered questions.

**Good output shape:**
- scenario labels and intent
- response template per scenario
- when to use public reply vs DM vs escalation
- tone guardrails
- follow-up actions for moderators
