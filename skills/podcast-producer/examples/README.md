# Podcast Producer — Example Output Shapes

Use these as patterns for podcast-production outputs.

## Example 1 — Weekly interview workflow

**Input:**
Build an operating workflow for a weekly founder interview show run by a host plus one contractor.

**Good output shape:**
| Stage | Owner | Deliverable | Deadline | Failure risk | Safeguard |
|---|---|---|---|---|---|
| Guest shortlist | producer | 6-name bench | 3 weeks out | thin pipeline | maintain rolling bench |
| Booking | producer | confirmed slot + intake form | 2 weeks out | calendar churn | hold backup guest |
| Prep brief | host | episode rundown + research notes | 2 days out | weak questions | standard prep template |
| Recording | host | clean raw track + backup | record day | tech failure | local + cloud backup |
| Edit + QA | editor | final cut | 3 days before publish | slow turnaround | locked revision window |
| Publish + package | producer | live episode + notes + clips | release day | sloppy metadata | pre-publish checklist |

Then add:
- recommended cadence and batch strategy
- what to automate vs keep manual
- first fixes if the team falls behind

## Example 2 — New show launch plan

**Input:**
Help us launch a B2B operations podcast in 6 weeks.

**Good output shape:**
- audience and listener promise
- recommended format and episode length
- trailer objective and content
- first 5 episode concepts with rationale
- production calendar by week
- publishing stack and ownership map
- launch-day distribution and repurposing plan
- top launch risks and mitigations

## Example 3 — Show audit

**Input:**
Our podcast quality swings wildly and downloads are flat. Diagnose it.

**Good output shape:**
| Problem | Likely root cause | Evidence to look for | Fix | Priority |
|---|---|---|---|---|
| episodes drag | poor rundown discipline | long tangents, weak edits | tighter segment timing + prep template | high |
| inconsistent release | no backlog buffer | missed dates after guest slips | create 2-episode buffer | high |
| weak discoverability | vague titles | low search/click response | shift to clearer title framework | medium |

Then add:
- 30-day stabilization plan
- what not to change yet

## Example 4 — Guest operations system

**Input:**
Create a guest workflow for our executive interview show.

**Good output shape:**
- qualification criteria
- outreach and scheduling sequence
- intake form fields
- prep packet contents
- recording-day checklist
- approvals / quote review policy
- no-show and reschedule contingencies
