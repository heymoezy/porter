# Storyboard Artist — Example Output Shapes

Use these patterns to produce boards that are readable, economical, and production-useful.

## Example 1 — 30-second product ad board

**Input:**
Turn this 30-second product ad script into a storyboard with panel purpose, framing, and transitions.

**Good output shape:**
- objective, runtime, and aspect-ratio assumptions
- 8–12 numbered panels
- for each panel: shot size, subject/action, narrative job, transition cue
- notes on product readability, pacing compression, and cheapest viable coverage
- 2–3 revision levers if budget or runtime changes

## Example 2 — Dialogue scene geography fix

**Input:**
Our two-character café scene feels visually confusing. Redesign the storyboard so power shifts and eyelines read clearly.

**Good output shape:**
- diagnosis of current confusion
- revised panel sequence with axis/geography logic
- notes on when to stay wide vs push in
- continuity hazards to watch on reverses and inserts
- optional stripped-down coverage plan for a faster shoot

## Example 3 — Explainer animation board

**Input:**
Create a storyboard for a 45-second explainer introducing a workflow product.

**Good output shape:**
- beat map tied to narration or on-screen text
- panel-by-panel visual progression from problem to solution to CTA
- transition logic and motion cues
- clarity notes on UI readability and abstraction level
- animatic recommendations for pacing the voiceover

## Example 4 — Action sequence revision memo

**Input:**
Review this chase sequence board and tell us what to cut, combine, or redraw.

**Good output shape:**
- top-line verdict on clarity and energy
- redundant panel clusters to remove
- missing orientation or bridge panels to add
- improved sequence order with reasons
- production-risk notes: resets, stunt complexity, edit gaps
