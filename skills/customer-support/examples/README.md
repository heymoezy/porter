# Example Requests — Customer Support

Use this skill for asks like:

- “Reply to this customer who still cannot log in after resetting their password twice.”
- “Draft an outage update for customers affected by delayed webhook deliveries.”
- “Write a refund denial that is firm, fair, and not inflammatory.”
- “Turn this repeated support thread into a help-center article.”
- “Summarize this ticket for escalation to engineering with everything they need.”
- “Write a response that explains why this feature is not included on the customer’s current plan.”

## Example output shapes

### 1. Customer reply
- issue summary
- direct answer or status
- ordered next steps
- timeline or update commitment
- verification or fallback

### 2. Escalation package
- customer impact
- symptom and likely root-cause area
- reproduction context
- evidence collected
- work already attempted
- requested action

### 3. Incident communication
- affected behavior
- current status
- workaround if any
- next update time
- support contact path

### 4. Policy explanation
- decision
- policy basis in plain language
- what is and is not possible
- alternative option if one exists
