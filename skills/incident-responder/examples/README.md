# Example Triggers — Incident Responder

Use this skill for requests like:

- “Help me run this production outage: API latency spiked after a deploy and customers cannot log in.”
- “Draft the incident channel updates and executive summary for a database failover gone wrong.”
- “Reconstruct a timeline from these alerts, deploy logs, and Slack messages.”
- “We think credentials may have been exposed—what should containment and communication look like?”
- “Turn this messy incident doc into a proper postmortem with real action items.”

## Strong outputs
- separate facts from hypotheses
- prioritize containment and customer impact clarity
- assign owners and next update times
- build a timestamped, evidence-backed timeline
- turn lessons into concrete prevention work
