# Media Strategist — Example Output Shapes

Use these as patterns for decision-ready media planning.

## Example 1 — Launch media plan

**Input:**
Plan a 90-day launch media strategy for a new AI bookkeeping product for small businesses.

**Good output shape:**
- objective and target audience
- channel-role table
- phase-by-phase flighting
- budget split by phase and channel
- creative requirements
- success metrics and test plan

## Example 2 — Channel-role grid

**Input:**
We need to balance paid search, YouTube, LinkedIn, podcasts, and email for a B2B category-creation push.

**Good output shape:**
| Channel | Primary role | Audience fit | Message depth | Budget posture | Success signal |
|---|---|---|---|---|---|
| YouTube | awareness + education | broad relevant reach | medium/high | core investment | branded search lift |
| paid search | demand capture | active intent | low | always-on | qualified conversions |
| email | nurture and conversion support | known audience only | high | support layer | engagement to pipeline |

Then add:
- reach vs frequency logic
- sequencing notes
- what not to expect from each channel

## Example 3 — Budget reallocation

**Input:**
A campaign is hitting CTR goals but pipeline is weak. Recommend how to rebalance media.

**Good output shape:**
- diagnosis of likely failure mode
- current-vs-recommended budget table
- channel role changes
- audience or message shifts
- measurement fixes
- 30-day verification plan

## Example 4 — Measurement framework

**Input:**
Design a measurement plan for a paid + creator + owned-media campaign.

**Good output shape:**
- role-by-channel metric map
- dashboard metrics vs causal metrics
- incrementality or lift-test recommendations
- attribution caveats
- decision rules for scale, hold, or cut
