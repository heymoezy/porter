# Animation Lead — Example Output Shapes

Use these as patterns for review and direction outputs.

## Example 1 — Shot review notes

**Input:**
Review a 20-second product animation for a SaaS landing page.

**Good output shape:**
- Goal: explain product flow clearly in 20s
- What works:
  - clean transitions between product states
  - good use of scale for hierarchy
- Blockers:
  1. intro move starts before user understands the initial screen
  2. camera drift competes with UI focus
  3. confirmation state lacks enough hold time
- Priority fixes:
  - add 6–10 frames hold before first transition
  - reduce camera motion during primary interaction
  - exaggerate success-state contrast and settle timing
- Polish:
  - unify easing family across secondary transitions
  - tighten logo outro spacing

## Example 2 — Blocking feedback

**Input:**
Give notes on a character blocking pass.

**Good output shape:**
- Shot 01: key poses are readable, but anticipation is too small to set up the action
- Shot 02: spacing feels even; add contrast so the impact lands
- Shot 03: silhouette collapses during turn; open torso/arm relationship
- Overall: blocking needs stronger pose contrast before spline work

## Example 3 — Animatic review

**Input:**
Review this explainer animatic.

**Good output shape:**
- story clarity
- pacing by beat
- transition smoothness
- shots that are redundant or underheld
- recommended runtime adjustments
- approval criteria before moving to animation

## Example 4 — Production plan

**Input:**
Plan the pipeline for a 60-second branded explainer.

**Good output shape:**
- stages:
  - script lock
  - storyboard
  - animatic
  - design/asset prep
  - animation passes
  - sound/composite/final
- review gates for each stage
- what must be approved before next stage
- likely risks and dependencies
