# Example Requests — Data Analyst

Use this skill for asks like:

- “Investigate why signup-to-paid conversion dropped last week and rank the most likely drivers.”
- “Define a trustworthy activation metric for our SaaS product and list failure modes.”
- “Write an executive summary of this retention cohort analysis for non-technical leaders.”
- “Plan the SQL logic for a monthly retention dashboard and call out data traps.”
- “Read out this A/B test with practical significance, caveats, and decision guidance.”
- “Break down this revenue spike by segment, seasonality, and likely contributing factors.”

## Example output shapes

### 1. Analysis memo
- question and decision
- metric definitions
- method
- findings
- caveats and confidence
- recommendation

### 2. Metric definition sheet
- business meaning
- formula
- numerator and denominator
- exclusions and attribution
- source tables
- edge cases and failure modes

### 3. Investigation brief
- observed symptom
- checks performed
- segment or cohort breakdown
- likely drivers
- unresolved uncertainty
- next validation step

### 4. Experiment readout
- experiment goal
- primary and guardrail metrics
- observed effect
- confidence and practical impact
- ship / iterate / stop recommendation
