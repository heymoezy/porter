# Proposal Writer — Example Output Shapes

Use these as patterns, not rigid templates.

## Example 1 — Client services proposal
**Input:** Create a proposal for redesigning a SaaS onboarding flow.

**Good output shape:**
- executive summary
- client problem and target outcome
- proposed workstreams and deliverables
- timeline with milestones
- investment and what it includes
- assumptions / exclusions
- next step: approve proposal and schedule kickoff

## Example 2 — Internal budget proposal
**Input:** Write a proposal for hiring two support specialists.

**Good output shape:**
- current pain and business impact
- recommended investment
- expected operational gains and risk reduction
- cost breakdown
- implementation timeline
- alternatives considered
- approval ask and review date

## Example 3 — Partnership pilot proposal
**Input:** Draft a pilot proposal for an integration partnership.

**Good output shape:**
| Section | What it covers |
|---|---|
| Opportunity | shared customer/problem and why now |
| Pilot scope | what both sides will ship or test |
| Responsibilities | owner by party |
| Success metrics | pilot win conditions |
| Risks | dependencies and fallback plan |
| Decision | signature, kickoff, or pilot approval |

## Example 4 — Strengthen a weak proposal
**Input:** This proposal sounds polished but vague.

**Good output shape:**
- key logic gaps
- rewritten executive summary
- tightened scope and exclusions
- clearer price / ask rationale
- stronger closing step
