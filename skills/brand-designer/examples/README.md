# Brand Designer — Example Output Shapes

Use these as patterns for strong brand-design deliverables.

## Example 1 — New identity direction

**Input:**
Create a visual direction for a climate-tech startup targeting enterprise buyers.

**Good output shape:**
- audience and positioning summary
- 2 to 3 design principles
- recommended logo, type, color, and imagery directions
- rationale tied to credibility and differentiation
- notes on how the identity should show up in product, sales, and web contexts

## Example 2 — Brand refresh critique

**Input:**
Our current brand feels inconsistent and generic. What should change?

**Good output shape:**
- diagnosis of current issues
- which existing assets to keep, revise, or retire
- prioritized brand-system fixes
- risks of over-correcting or losing recognition
- next-step guideline or rollout recommendation

## Example 3 — Brand guideline skeleton

**Input:**
Outline a brand book for our team.

**Good output shape:**
- purpose of the brand system
- logo rules and misuse examples
- color roles and accessibility notes
- type hierarchy and spacing guidance
- imagery, icon, or illustration rules
- examples of application across common touchpoints

## Example 4 — Sub-brand architecture

**Input:**
We need a visual system for our product line without breaking the master brand.

**Good output shape:**
- architecture model recommendation
- what stays constant versus what varies
- naming and visual differentiation logic
- risks of fragmentation
- rollout guidance for templates and teams

## Example 5 — Identity concept comparison

**Input:**
Give us three brand directions for a premium hospitality company.

**Good output shape:**
- concept names or themes
- distinctive visual characteristics of each
- strategic pros and tradeoffs
- where each concept would work best
- recommendation with rationale
