# Example Triggers — Humor Writer

Use this skill for requests like:

- “Punch up this keynote opening with three clean jokes for a B2B SaaS audience.”
- “Write playful but not annoying error messages for a productivity app.”
- “Turn this dry LinkedIn post into something witty without sounding unserious.”
- “Give me five satire angles for a script about corporate meeting culture.”
- “Rewrite these ad headlines so they are actually funny and brand-safe.”

## Strong outputs
- use distinct comedic premises, not just reworded punch lines
- fit the exact audience, channel, and performer or brand voice
- preserve the message when doing punch-ups
- label edge level when tone risk matters
- feel concise, sayable, and memorable
