# Design Critic — Example Output Shapes

Use these patterns to keep critique sharp and decision-ready.

## Example 1 — Dashboard critique

**Input:**
Review this analytics dashboard homepage.

**Good output shape:**
- Intended job:
  - orient the user quickly and surface the most important next action
- What works:
  - strong summary strip
  - restrained color palette
- Highest-impact problems:
  - too many equal-weight modules
  - weak scan path after the header
  - sidebar competes with primary content
- Specific changes:
  - collapse low-frequency widgets
  - create one dominant center-of-gravity module
  - demote navigational chrome

## Example 2 — Onboarding-flow diagnosis

**Input:**
Why does our onboarding feel confusing?

**Good output shape:**
- main mental-model breaks
- steps that ask for commitment before confidence exists
- unnecessary branching or copy load
- recommended simplifications
- tradeoffs of shortening versus educating more

## Example 3 — Two-concept comparison

**Input:**
Pick between these two landing page directions.

**Good output shape:**
- evaluation rubric:
  - clarity
  - trust
  - differentiation
  - conversion readiness
- winner and why
- what to borrow from the losing concept
- what to refine before launch

## Example 4 — Brand drift review

**Input:**
Our new campaign visuals feel off-brand. Diagnose it.

**Good output shape:**
- core mismatch summary
- where typography, imagery, or color language drifted
- what must reset to regain recognition
- what fresh elements can stay without breaking the brand

## Example 5 — Mobile screen critique

**Input:**
Critique this mobile checkout screen.

**Good output shape:**
- job of the screen
- confidence-damaging moments
- hierarchy and copy issues near the payment decision
- concrete fixes to reduce hesitation and error risk
