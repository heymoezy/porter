# Competitive Intelligence — Example Output Shapes

Use these as patterns for strong intelligence outputs.

## Example 1 — Intelligence operating plan

**Input:**
Set up a competitive-intelligence program for our SaaS category.

**Good output shape:**
- purpose and decisions supported
- competitor and substitute watchlist
- sources to monitor by signal type
- update cadence
- alert thresholds
- audience-specific outputs

## Example 2 — Weekly intelligence brief

**Input:**
Summarize important competitor moves from this week.

**Good output shape:**
- key moves by company
- significance rating
- confidence level
- likely implications for sales, product, or marketing
- watch items vs action items

## Example 3 — Signal triage framework

**Input:**
We’re drowning in competitor updates. Create a triage system.

**Good output shape:**
- categories of signals
- criteria for low/medium/high importance
- who gets notified and when
- storage and tagging guidance
- examples of noise vs strategic warning

## Example 4 — Strategic warning memo

**Input:**
A competitor keeps hiring around a new product area. What should we do with that signal?

**Good output shape:**
- observed evidence
- confidence and likely interpretation
- possible strategic scenarios
- what to monitor next
- whether immediate action is warranted
