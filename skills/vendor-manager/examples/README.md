# vendor-manager — Example Output Shapes

Use these as patterns for decision-ready vendor work.

## Example 1 — SaaS shortlist

**Input:**
Compare three help-desk platforms for a 40-person startup with SOC 2 requirements, limited migration capacity, and a hard budget cap.

**Good output shape:**
- Recommendation: shortlist two, eliminate one early
- Weighted criteria table: security, migration effort, workflow fit, reporting, support, TCO
- Why the winner leads
- Main implementation and renewal risks
- Next step: demo script or pilot plan

## Example 2 — Renewal negotiation brief

**Input:**
Our analytics vendor wants a 22% renewal increase. Prepare negotiation points.

**Good output shape:**
- Renewal posture: renew only if terms improve
- Must-win items: price cap, seat true-down, better support SLA, usage transparency
- Givebacks: longer term, reference call, limited prepay
- Walk-away triggers
- BATNA / fallback option

## Example 3 — Underperforming agency review

**Input:**
Assess whether we should keep or replace a paid-media agency after two poor quarters.

**Good output shape:**
- Verdict: remediate for one quarter or replace now
- Evidence summary: delivery misses, reporting quality, learning velocity, CAC trend
- Risks of staying vs switching
- 30-day remediation scorecard or transition plan

## Example 4 — Vendor-governance scorecard

**Input:**
Set a quarterly review framework for our 3PL partner.

**Good output shape:**
| KPI | Target | Current issue | Owner | Escalation trigger |
|---|---|---|---|---|
| on-time delivery | ... | ... | ... | ... |
| damage rate | ... | ... | ... | ... |
| invoice accuracy | ... | ... | ... | ... |

Then add:
- review cadence
- escalation path
- rebid trigger conditions
