# Bibliographic Researcher — Example Output Shapes

Use these as patterns for strong bibliographic deliverables.

## Example 1 — Annotated starter bibliography

**Input:**
Build a starter bibliography on AI model evaluation for product teams.

**Good output shape:**
- scope and exclusions
- 8 to 15 high-signal sources
- grouped by standards, benchmarks, practical guidance, and critiques
- short annotation per source
- notes on gaps or follow-up areas

## Example 2 — Thematic academic bibliography

**Input:**
Create an annotated bibliography on urban heat island mitigation strategies.

**Good output shape:**
- databases or search approach used
- thematic sections such as tree canopy, cool roofs, pavements, and policy
- citation plus 2 to 4 sentence annotation per source
- method or evidence-type notes
- limitations and under-covered regions

## Example 3 — Source landscape for a memo

**Input:**
Find credible sources for a memo about remote work productivity.

**Good output shape:**
- source buckets: meta-analyses, employer studies, surveys, critiques
- strongest sources highlighted first
- notes on methodological weaknesses in survey-heavy evidence
- recommended citations to use in the memo
- sources to avoid relying on heavily

## Example 4 — Citation cleanup pass

**Input:**
Normalize and de-duplicate this messy bibliography.

**Good output shape:**
- duplicate groups identified
- corrected canonical citations
- missing metadata flagged
- formatting inconsistencies fixed
- unresolved verification issues listed separately

## Example 5 — Evidence gap scan

**Input:**
Map the literature on adolescent sleep interventions and tell me where the gaps are.

**Good output shape:**
- scope definition and inclusion logic
- bibliography grouped by intervention type
- indication of study design strength
- clear under-researched populations or settings
- suggested search expansions for the next pass
