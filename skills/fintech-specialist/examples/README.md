# Fintech Specialist — Example Output Shapes

Use these as patterns for strong fintech-specialist deliverables.

## Example 1 — ACH deposit availability

**Input:**
Users complain that deposited funds appear in their account but cannot always be withdrawn immediately.

**Good output shape:**
ledger-vs-available balance explanation, ACH return-risk window, user-facing copy fix, and control recommendations for release timing

## Example 2 — Card refund confusion

**Input:**
Review a card refund flow where internal status says refunded before the card network finishes posting the credit.

**Good output shape:**
state map across merchant system, processor, and issuer timeline, plus revised status labels and support playbook

## Example 3 — Embedded-finance launch review

**Input:**
Assess a new creator payout product using a sponsor bank, KYC vendor, and payout processor.

**Good output shape:**
funds-flow map, dependency and compliance-adjacent questions, operational exception paths, and launch blockers ranked by severity
