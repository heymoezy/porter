# Mergers Analyst — Example Output Shapes

## Example 1 — Deal screen

**Input:**
Should a vertical SaaS company acquire a services-heavy implementation partner?

**Good output shape:**
- buyer objective
- target fit with product and customer strategy
- what capability is actually being bought
- main risks and dis-synergies
- recommendation with walk-away triggers

## Example 2 — Diligence tracker

**Input:**
Summarize confirmatory diligence findings for a Series B target.

**Good output shape:**
| Issue | Evidence | Category | Deal impact | Recommended response |
|---|---|---|---|---|
| Top 3 customers = 48% of ARR | customer revenue review | price-adjuster | concentration risk widens downside case | lower base multiple / add retention protection |
| Founder owns key customer relationships | management interviews | integration-critical | churn risk post-close | retention plan and earnout linkage |

## Example 3 — Valuation range

**Input:**
Frame a valuation range for a target with strong growth but weak gross margins.

**Good output shape:**
- base / downside / upside cases
- QoE adjustments and margin normalization assumptions
- synergy treatment by scenario
- price range
- conditions required for the top end to be defensible

## Example 4 — Buy vs build

**Input:**
Should we buy a small AI workflow company or build the feature internally?

**Good output shape:**
| Option | Speed | Capability gap closed | Execution risk | Integration or org cost | Recommendation |
|---|---|---|---|---|---|
