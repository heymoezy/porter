# Editorial Planner — Example Deliverable Shapes

Use these patterns to keep planning outputs strategic and buildable.

## 1. Quarterly editorial calendar

**Input:**
Plan next quarter for a B2B fintech content team.

**Strong shape:**
- quarter thesis
- audience segments and key tensions
- 3–5 pillars with explicit jobs
- month-by-month arcs or themes
- priority pieces with format, role, and dependencies
- what gets cut if capacity tightens

## 2. Product launch sequence

**Input:**
Build the content plan for a new feature launch.

**Strong shape:**
- pre-launch education plan
- launch-day assets
- follow-up proof, FAQ, and objection-handling pieces
- blog / newsletter / social sequencing
- owner and dependency notes
- success signals to watch

## 3. Backlog triage

**Input:**
We have 120 ideas. Tell us what survives.

**Strong shape:**
- scoring or triage logic
- keep / merge / defer / kill sections
- patterns in what is overrepresented
- top-priority ideas rewritten as strong angles
- proof or gap areas still missing
- next briefs to commission

## 4. Recurring series design

**Input:**
Design a weekly series for founder operators.

**Strong shape:**
- audience promise
- repeatable format and episode structure
- sample installments
- sourcing workflow and effort estimate
- failure modes and kill criteria

## 5. Multi-channel editorial system

**Input:**
Define how blog, newsletter, and social should work together.

**Strong shape:**
- job of each channel
- primary-to-secondary sequencing logic
- reuse versus native creation guidance
- cadence recommendation
- editorial guardrails that keep the voice coherent
