# Content Writer — Example Output Shapes

Use these as patterns for practical drafting tasks.

## Example 1 — Blog draft

**Input:**
Write a blog post explaining why our analytics product is moving to real-time dashboards.

**Good output shape:**
- strong title
- concise introduction with user problem
- 3–5 clear sections
- examples or practical implications
- conclusion with CTA or next step

## Example 2 — Landing page section copy

**Input:**
Draft homepage copy for a B2B workflow automation product.

**Good output shape:**
- hero headline and subhead
- proof/benefit sections
- objection-handling section
- CTA copy
- optional alternate headline options

## Example 3 — Documentation rewrite

**Input:**
Rewrite this messy setup guide so new users can follow it.

**Good output shape:**
- short overview
- prerequisites
- numbered steps
- troubleshooting or common mistakes
- concise closing next step

## Example 4 — Internal announcement

**Input:**
Draft a team note announcing the new support escalation process.

**Good output shape:**
- clear purpose statement
- what is changing
- who is affected
- effective date and action required
- contact path for questions
