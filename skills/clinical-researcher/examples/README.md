# Clinical Researcher — Example Output Shapes

Use these as patterns for clinically rigorous outputs.

## Example 1 — Study design recommendation

**Input:**
We want to study whether a digital therapeutic reduces migraine days.

**Good output shape:**
- research question
- recommended design and why
- target population
- primary and secondary endpoints
- follow-up duration
- bias/safety/feasibility considerations
- what would make the evidence credible

## Example 2 — Protocol critique

**Input:**
Review this trial outline for weaknesses.

**Good output shape:**
- strengths
- key protocol gaps
- endpoint problems
- inclusion/exclusion concerns
- safety monitoring concerns
- data integrity or feasibility issues
- highest-priority fixes before approval

## Example 3 — Evidence interpretation memo

**Input:**
Interpret these observational findings on a new oncology biomarker.

**Good output shape:**
- summary of findings
- design limitations
- confounding and selection-bias risks
- what can be claimed confidently
- what remains hypothesis-generating only
- recommended follow-up evidence

## Example 4 — Endpoint framework

**Input:**
Help define endpoints for a diabetes study.

**Good output shape:**
- primary endpoint options
- secondary and safety endpoints
- clinical vs surrogate outcome discussion
- measurement timing
- data collection considerations
- recommendation with rationale
