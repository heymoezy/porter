# Franchise Analyst — Example Output Shapes

Use these as patterns for strong franchise-analyst deliverables.

## Example 1 — New franchise screen

**Input:**
Assess whether a fast-casual concept is attractive for a first-time owner-operator in a high-rent metro market.

**Good output shape:**
investment and fee-stack breakdown, realistic mature-unit economics, labor and rent sensitivity, operator-fit view, and top diligence questions

## Example 2 — Network health review

**Input:**
Explain why a franchise brand keeps adding outlets while franchisee satisfaction appears weak.

**Good output shape:**
analysis of closures, transfers, support quality, same-store economics, and franchisor-versus-franchisee incentive mismatch

## Example 3 — Second-unit expansion

**Input:**
Review whether an existing franchisee should open a second store in an adjacent territory.

**Good output shape:**
territory overlap and management-bandwidth analysis, downside scenario economics, and a proceed-only-if checklist
