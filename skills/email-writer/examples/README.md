# Email Writer — Example Output Shapes

Use these patterns to shape strong email deliverables.

## Example 1 — Internal decision request

**Input:**
Draft an email asking engineering leadership to confirm whether we are still committing to Friday's release.

**Good output shape:**
- 2-4 subject line options
- one-sentence purpose in the opening
- concise status / risk context
- explicit decision needed and deadline
- short close

## Example 2 — Customer apology with next steps

**Input:**
Write an email to a customer whose issue is still unresolved after multiple follow-ups.

**Good output shape:**
- acknowledge impact without defensiveness
- state current status honestly
- explain what happens next and by when
- identify owner / point of contact
- calm close that preserves trust

## Example 3 — Follow-up after no response

**Input:**
Write a follow-up email to a prospect who never replied to my original message.

**Good output shape:**
- brief reminder of context
- restated value in one or two lines
- easy reply path, ideally yes/no or short-answer
- respectful exit ramp

## Example 4 — Scope boundary email

**Input:**
Help me tell a client their new requests are out of scope and need a separate estimate.

**Good output shape:**
- acknowledgment of the request
- direct boundary statement
- brief rationale if useful
- options / next steps
- firm but non-inflammatory close

## Example 5 — Leadership update

**Input:**
Summarize this week's project status for the executive team.

**Good output shape:**
- immediate top-line status
- bullets for progress, risks, and decisions needed
- explicit owners / asks where relevant
- tight length appropriate for exec scanning

## Example 6 — Sensitive internal reset

**Input:**
Draft an email to a teammate whose missed deadlines are hurting the project.

**Good output shape:**
- respectful acknowledgment of the issue
- concrete examples or impact
- expectation reset / next steps
- collaborative but clear tone
