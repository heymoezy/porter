# Academic Researcher — Example Output Shapes

Use these as patterns for structure and rigor.

## Example 1 — Literature review memo

**Input:**
Review the literature on whether remote work improves productivity.

**Good output shape:**
- Research question and scope
- Search approach:
  - date range
  - source classes
  - search terms
  - inclusion/exclusion logic
- Evidence base summary:
  - number and type of sources
- Thematic findings:
  - productivity effects vary by task type
  - self-reported productivity differs from observed metrics
  - management practices moderate outcomes
- Areas of disagreement:
  - knowledge work vs collaborative work
  - short-term vs long-term studies
- Limitations:
  - publication bias
  - industry concentration
  - inconsistent productivity definitions
- Conclusion:
  - strongest evidence suggests conditional, context-dependent gains

## Example 2 — Annotated bibliography

**Input:**
Build an annotated bibliography on AI in medical education.

**Good output shape:**
For each source:
- full citation or verification-ready source details
- source type
- research question
- method/design
- key finding
- limitation
- why it matters for the user’s question

## Example 3 — Evidence comparison table

**Input:**
Compare papers on the effectiveness of retrieval practice in learning.

**Good output shape:**
| Source | Study type | Context/sample | Main finding | Limitations | Confidence |
|---|---|---|---|---|---|
| Source A | Meta-analysis | K-12 + university | Positive effect overall | Publication bias possible | High |
| Source B | RCT | Undergraduate psych students | Improved retention after delay | Narrow sample | Medium |

Then add:
- synthesis paragraph
- transferability notes
- gaps in literature

## Example 4 — Research protocol

**Input:**
Plan a systematic-style review on carbon labeling and consumer behavior.

**Good output shape:**
- review objective
- primary question
- inclusion criteria
- exclusion criteria
- proposed databases/source sets
- draft search string logic
- screening process
- extraction fields
- risk-of-bias considerations
- expected limits and timeline
