# QA Checklist — Memory Curator

- Output preserves durable signal instead of transcript exhaust.
- Memory type fits the information shape.
- Facts, inferences, proposals, and directives are clearly separated.
- Candidate text is concise, concrete, and retrieval-friendly.
- Duplicates, conflicts, and superseded entries are handled intentionally.
- Uncertain or time-sensitive claims are labeled with confidence or recency.
- Rejected items are rejected for a stated reason, not silently omitted.
