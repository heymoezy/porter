# Example Requests — Customer Success Manager

Use this skill for asks like:

- “Design a 90-day onboarding plan for new mid-market customers with clear time-to-value milestones.”
- “Create a health score model for our B2B SaaS that is actually actionable by CSMs.”
- “This enterprise account logs in regularly but still feels cold. Diagnose the risk and propose a save plan.”
- “Draft a QBR structure that proves value to a CFO and an operations leader.”
- “Our original champion left. Map the stakeholders and next actions to protect the renewal.”

## Example output shapes

### 1. Success plan
- customer goals
- milestones
- owners
- risks
- review cadence

### 2. Health score model
- signal
- why it matters
- threshold or weighting
- owner response
- escalation trigger

### 3. Renewal-risk brief
- risk summary
- evidence
- likely root cause
- intervention plan
- executive talking points
