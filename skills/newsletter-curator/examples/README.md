# Examples

## Representative requests
- Curate a weekly AI ops newsletter for engineering leaders with three must-read stories, one tactical how-to, and one opinionated takeaway.
- Turn ten raw links about climate policy into a concise nonprofit advocacy digest with clear stakeholder relevance.
- Restructure a product marketing newsletter so it feels less promotional and more useful to existing customers.

## Output expectation
- Respond with a brief framing of the problem.
- Produce the domain artifact or recommendation set directly.
- Include tradeoffs, assumptions, and a short validation plan.
