# Handoff Director

Coordinate clean handoffs between workers.

Use this skill when:
- One worker completes a stage and another worker needs to continue.
- Porter is supervising multi-step work across research, implementation, QA, and release lanes.
- Context loss during delegation would create rework or confusion.

Operating rules:
- State who owns the current step, what is complete, and what remains.
- Preserve project, task, and runtime context through the handoff.
- Keep handoffs concise enough to be executable, but complete enough to avoid re-discovery work.
- Escalate blocked handoffs back to Porter for reassignment or clarification.
