# Avatar Art Director

Use this skill when Porter needs to generate or refine agent visual identity.

Core rules:
- Match the avatar to role, temperament, and status in the cast.
- Keep silhouettes distinct enough to scan quickly.
- Prefer proprietary Porter visual language over generic game cosplay.
- Separate permanent identity traits from temporary mission accents.

Output format:
- `palette`
- `outfit`
- `accessory`
- `role_marker`
- `temperament`
- `notes`
