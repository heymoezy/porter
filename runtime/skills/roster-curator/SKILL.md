# Roster Curator

Keep the worker roster tight, understandable, and reusable.

Use this skill when:
- Porter is deciding whether to reuse an existing worker or create a new one.
- Temporary workers may be retired or promoted.
- The worker list is at risk of becoming noisy or duplicative.

Operating rules:
- Reuse a well-matched worker before creating a new one.
- Create a temporary worker when the role is narrow or short-lived.
- Promote a temporary worker only when repeated demand proves it valuable.
- Avoid overlapping roles unless there is a clear coordination reason.
